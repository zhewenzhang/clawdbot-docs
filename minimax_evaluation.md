# MiniMax 模型评估计划

## 评估目标
验证 MiniMax-M2.1 在各类任务上的表现，建立最佳实践

## 评估维度

### 1. 中文理解
- [ ] 新闻摘要准确性
- [ ] 技术文档理解
- [ ] 口语化指令解析

### 2. 代码生成
- [ ] Python 脚本编写
- [ ] Shell 命令生成
- [ ] 代码注释和解释

### 3. 逻辑推理
- [ ] 数据分析洞察
- [ ] 因果关系判断
- [ ] 多步骤任务规划

### 4. 创意写作
- [ ] 报告排版质量
- [ ] 语言流畅度
- [ ] 格式规范性

### 5. 工具调用
- [ ] 文件操作准确性
- [ ] API 调用成功率
- [ ] 错误处理能力

## 测试方法

每个维度执行 3 个小任务，记录：
- 响应质量 (1-5)
- Token 消耗
- 响应时间
- 改进建议

## 预期结果
生成 MiniMax 最佳使用指南，优化任务分配策略。
