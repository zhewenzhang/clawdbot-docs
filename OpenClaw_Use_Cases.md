## 1. 半导体行业案例

### 案例1：每日行业新闻汇总

**场景**: 每天早上8点自动汇总半导体行业新闻

```
使用技能：
├── cron - 定时触发
├── blogwatcher - 抓取行业博客
├── summarize - 总结新闻要点
└── gog - 发送到Gmail

工作流程：
1. 7:59 cron触发任务
2. blogwatcher抓取最新新闻
3. summarize总结每条新闻
4. gog发送到邮箱
5. 8:00收到汇总邮件

效果：
- 节省30分钟/天
- 不遗漏重要新闻
- 信息质量高
```

**技能组合**: ⭐⭐⭐⭐⭐  
**时间节省**: 30分钟/天  
**实用价值**: 极高
