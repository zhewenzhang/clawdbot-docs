## 三、库存周期深度分析

### 3.1 库存周期定义

**库存周期 = 产业链库存调整周期**

```
库存周期示意图

   库存水位
      ↑
  高  │    ┌─┐         ┌─┐         ┌─┐
      │    │ │         │ │         │ │
      │    │ │         │ │         │ │
  低  │────┘ └─────────┘ └─────────┘ └────
      └─────────────────────────────────→ 时间
      
      周期：1-2 年
```

### 3.2 库存周期阶段

| 阶段 | 库存特征 | 芯片需求 | 投资策略 |
|------|----------|----------|----------|
| **主动补库** | 库存低位+需求回升 | 快速增长 | 增持 |
| **被动补库** | 库存回升+需求见顶 | 增速放缓 | 持有 |
| **主动去库** | 库存高位+需求下滑 | 负增长 | 减持 |
| **被动去库** | 库存下降+需求回升 | 增速转正 | 增持 |

### 3.3 库存周期指标

| 指标 | 计算方法 | 含义 |
|------|----------|------|
| **库存天数** | 平均库存/日均销售 | 库存水平 |
| **库存周转率** | 销售成本/平均库存 | 周转速度 |
| **库存同比** | 库存 YoY | 变化趋势 |
| **库销比** | 库存/销售额 | 库存压力 |

```

                    │
                    ↓
              底部期 (布局)

         ┌────────┬────────┬────────┬────────┐
         │ 底部期 │ 扩张期 │ 高峰期 │ 去库期 │
├────────┼────────┼────────┼────────┼────────┤
│ 股票   │ 布局   │ 增持   │ 持有   │ 减持   │

│ 现金   │ 低配   │ 低配   │ 低配   │ 高配   │
└────────┴────────┴────────┴────────┴────────┘
```
