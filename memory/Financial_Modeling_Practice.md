## 二、Top-down 营收建模

### 2.1 Top-down 方法定义

**Top-down = 自上而下**

```
Top-down 建模逻辑

┌─────────────────────────────────────────┐
│          全球市场（总TAM）               │
│              $1000B                     │
└──────────────────┬──────────────────────┘
                   ▼
┌─────────────────────────────────────────┐
│          中国市场（SAM）                 │
│               $300B                     │
└──────────────────┬──────────────────────┘
                   ▼
┌─────────────────────────────────────────┐
│         公司目标市场（SOM）              │
│                $30B                     │
└──────────────────┬──────────────────────┘
                   ▼
┌─────────────────────────────────────────┐
│           公司营收预测                   │
│               $5B                       │
└─────────────────────────────────────────┘

TAM = Total Addressable Market（总市场）
SAM = Serviceable Available Market（可服务市场）
SOM = Serviceable Obtainable Market（可获得市场）
```

### 2.2 Top-down 建模步骤

**步骤 1：确定 TAM（总市场）**

| 数据来源 | 方法 | 优缺点 |
|----------|------|--------|
| **行业协会** | 引用权威数据 | 权威但滞后 |
| **咨询公司** | 购买报告 | 专业但昂贵 |
| **公开信息** | 估算 | 灵活但可能不准 |
| **公司披露** | 参考招股书 | 方便但需验证 |

**步骤 2：确定 SAM（可服务市场）**

```
SAM = TAM × 目标市场占比

示例：
- TAM：全球半导体市场 $500B
- 中国市场占比：30%
- 目标产品线占比：20%
- SAM = $500B × 30% × 20% = $30B
```

**步骤 3：确定 SOM（可获得市场）**

```
SOM = SAM × 可获得市场份额

示例：
- SAM：$30B
- 公司当前份额：10%
- 3 年目标份额：15%
- SOM = $30B × 15% = $4.5B
```

**步骤 4：验证 SOM 合理性**

| 验证方法 | 说明 |
|----------|------|
| **历史增速对比** | 公司增速 vs 市场增速 |
| **竞争对手对比** | 份额变化趋势 |
| **产能对比** | 是否支持目标收入 |
| **管理层指引** | 管理层预期验证 |

### 2.3 Top-down 建模示例

**案例：某 IC 载板公司**

```
Top-down 建模

1. TAM（全球 IC 载板市场）
   - 2024 年：$12.6B
   - 2029 年预测：$20.1B
   - CAGR：9.8%

2. SAM（ABF 载板市场）
   - IC 载板中占比：40%
   - 2024 年 ABF 市场：$5.0B

3. SOM（公司目标市场）
   - ABF 载板中占比：20%
   - 2024 年 SOM：$1.0B

4. 营收预测
   - 2024E：$1.0B
   - 2025E：$1.2B（+20%）
   - 2026E：$1.4B（+17%）
```

## 三、Bottom-up 营收建模

### 3.1 Bottom-up 方法定义

**Bottom-up = 自下而上**

```
Bottom-up 建模逻辑

┌─────────────────────────────────────────┐
│       产能（产能利用率）                 │
│              100K/月                    │
└──────────────────┬──────────────────────┘
                   ▼
┌─────────────────────────────────────────┐
│       产能利用率                         │
│               85%                       │
└──────────────────┬──────────────────────┘
                   ▼
┌─────────────────────────────────────────┐
│       产出量（85K/月 × 12 = 1,020K）     │
│            1,020K/年                    │
└──────────────────┬──────────────────────┘
                   ▼
┌─────────────────────────────────────────┐
│       ASP（平均售价）                    │
│              $1000/片                   │
└──────────────────┬──────────────────────┘
                   ▼
┌─────────────────────────────────────────┐
│       营收预测                           │
│         $1.02B                          │
└─────────────────────────────────────────┘
```

### 3.2 Bottom-up 建模步骤

**步骤 1：产能预测**

| 项目 | 2024 | 2025 | 2026 |
|------|------|------|------|
| **设计产能** | 100K/月 | 120K/月 | 150K/月 |
| **产能利用率** | 85% | 90% | 92% |
| **有效产能** | 85K/月 | 108K/月 | 138K/月 |
| **年产能** | 1,020K | 1,296K | 1,656K |

**步骤 2：ASP 预测**

| 项目 | 2024 | 2025 | 2026 |
|------|------|------|------|
| **产品结构** | 70% 高端 | 75% 高端 | 80% 高端 |
| **高端 ASP** | $1,200 | $1,250 | $1,300 |
| **低端 ASP** | $800 | $820 | $840 |
| **加权 ASP** | $1,080 | $1,143 | $1,208 |

**步骤 3：营收计算**

| 项目 | 2024 | 2025 | 2026 |
|------|------|------|------|
| **销量（K）** | 1,020 | 1,296 | 1,656 |
| **ASP** | $1,080 | $1,143 | $1,208 |
| **营收（$M）** | $1,102 | $1,481 | $2,001 |

### 3.3 Bottom-up 建模示例

**案例：某晶圆代工公司**

```
Bottom-up 建模

1. 产能规划
   - 2024 年底：100K/月（12 寸）
   - 2025 年底：130K/月
   - 2026 年底：160K/月

2. 产能利用率
   - 2024 年：75%（周期底部）
   - 2025 年：85%
   - 2026 年：90%

3. 晶圆 ASP
   - 2024 年：$1,500（成熟制程为主）
   - 2025 年：$1,600（28nm 占比提升）
   - 2026 年：$1,700（14nm 占比提升）

4. 营收计算
   - 2024E：100K × 75% × 12 × $1,500 = $1.35B
   - 2025E：130K × 85% × 12 × $1,600 = $2.12B
   - 2026E：160K × 90% × 12 × $1,700 = $2.93B

5. 营收预测
   - 2024E：$1.35B（+15% YoY）
   - 2025E：$2.12B（+57% YoY）
   - 2026E：$2.93B（+38% YoY）
```

## 四、建模方法对比

### 4.1 方法对比

| 维度 | Top-down | Bottom-up |
|------|----------|-----------|
| **逻辑** | 从市场到公司 | 从公司到市场 |
| **数据需求** | 市场规模数据 | 公司运营数据 |
| **优点** | 考虑市场空间 | 贴近公司实际 |
| **缺点** | 可能脱离公司能力 | 可能忽视市场限制 |
| **适用** | 新市场、渗透率分析 | 成熟公司、产能分析 |
| **验证** | 需交叉验证 | 需交叉验证 |

### 4.2 交叉验证

```
交叉验证逻辑

Top-down 预测营收 = $1.0B
Bottom-up 预测营收 = $1.1B

差异：10%（合理范围）

差异分析：
- Top-down 可能高估：公司能力不足
- Bottom-up 可能低估：市场空间更大

解决方案：
- 取中值或加权平均
- 分析差异原因
- 调整假设重新计算
```

## 五、成本结构建模

### 5.1 成本分类

```
成本结构

┌─────────────────────────────────────────┐
│              总成本                      │
├─────────────────┬───────────────────────┤
│     固定成本    │      变动成本          │
├─────────────────┼───────────────────────┤
│  • 折旧摊销    │  • 直接材料            │
│  • 人工成本    │  • 直接人工            │
│  • 研发费用    │  • 制造费用            │
│  • 管理费用    │  • 水电能源            │
└─────────────────┴───────────────────────┘
```

### 5.2 成本建模示例

**案例：某封测公司**

| 成本项目 | 占比 | 2024E | 2025E | 2026E |
|----------|------|-------|-------|-------|
| **直接材料** | 45% | $450M | $540M | $630M |
| **直接人工** | 20% | $200M | $240M | $280M |
| **制造费用** | 15% | $150M | $180M | $210M |
| **折旧摊销** | 12% | $120M | $144M | $168M |
| **管理费用** | 5% | $50M | $60M | $70M |
| **研发费用** | 3% | $30M | $36M | $42M |
| **合计** | 100% | $1,000M | $1,200M | $1,400M |

### 5.3 毛利率预测

```
毛利率 = 1 - 成本/营收

示例：
- 2024E 营收：$1,100M
- 2024E 成本：$1,000M
- 2024E 毛利率：1 - 1000/1100 = 9.1%

毛利率变化分析：
- 规模效应：固定成本摊薄，毛利率提升
- 产品结构：高端产品占比提升，毛利率提升
- 产能利用率：利用率提升，毛利率提升
```

## 六、建模实战案例

### 6.1 案例：某 IC 载板公司

**Top-down 建模**

```
1. TAM（全球 IC 载板市场）
   - 2024 年：$12.6B
   - 2029 年：$20.1B
   - CAGR：9.8%

2. ABF 载板占比
   - IC 载板中占比：40%
   - ABF 市场：$5.0B

3. 公司目标
   - ABF 份额目标：20%
   - SOM：$1.0B

4. 营收预测（Top-down）
   - 2024E：$1.0B
   - 2025E：$1.15B
   - 2026E：$1.32B
```

**Bottom-up 建模**

```
1. 产能规划
   - 2024 年：100K/月
   - 2025 年：120K/月
   - 2026 年：150K/月

2. 产能利用率
   - 2024 年：85%
   - 2025 年：90%
   - 2026 年：92%

3. ASP
   - 2024 年：$1,000/片
   - 2025 年：$1,050/片
   - 2026 年：$1,100/片

4. 营收预测（Bottom-up）
   - 2024E：100K × 85% × 12 × $1,000 = $1.02B
   - 2025E：120K × 90% × 12 × $1,050 = $1.36B
   - 2026E：150K × 92% × 12 × $1,100 = $1.82B
```

**交叉验证与调整**

```
差异分析：
- Top-down 2024E：$1.0B
- Bottom-up 2024E：$1.02B
- 差异：2%（合理）

最终营收预测（取平均）：
- 2024E：$1.01B
- 2025E：$1.26B
- 2026E：$1.57B
```

## 七、建模最佳实践

### 7.1 建模原则

| 原则 | 说明 |
|------|------|
| **假设透明** | 所有假设需明确记录 |
| **可追溯** | 数据来源需可追溯 |
| **可验证** | 模型结果需可验证 |
| **灵活性** | 模型需易于调整 |
| **简洁性** | 避免过度复杂 |

### 7.2 常见错误

| 错误 | 避免方法 |
|------|----------|
| **假设过于乐观** | 保守假设，敏感性分析 |
| **忽视风险** | 情景分析，风险调整 |
| **数据不实** | 交叉验证，多源验证 |
| **模型复杂** | 简化模型，聚焦关键 |
| **不更新** | 定期更新模型 |

### 7.3 模型检查清单

| 检查项 | 说明 |
|--------|------|
| **假设合理性** | 假设是否合理？ |
| **逻辑一致性** | 各部分是否一致？ |
| **数据准确性** | 数据是否准确？ |
| **结果验证** | 结果是否合理？ |
| **敏感性** | 关键假设敏感？ |

## 八、学习要点总结

### 8.1 核心方法

| 方法 | 逻辑 | 适用场景 |
|------|------|----------|
| **Top-down** | 市场 → 份额 → 营收 | 新市场、渗透率 |
| **Bottom-up** | 产能 → 利用率 → ASP → 营收 | 成熟公司、产能驱动 |
| **交叉验证** | 两种方法对比验证 | 所有建模 |

### 8.2 关键假设

| 假设类型 | 示例 | 数据来源 |
|----------|------|----------|
| **市场规模** | TAM、SAM、SOM | 行业协会、咨询公司 |
| **市场份额** | 当前份额、目标份额 | 公司历史、竞争对手 |
| **产能** | 设计产能、利用率 | 公司公告、行业数据 |
| **ASP** | 产品价格、价格趋势 | 公司数据、市场数据 |
| **成本** | 成本结构、成本趋势 | 公司数据、行业数据 |
