## 📚 七、核心术语表

### 市场术语

| 术语 | 含义 |
|------|------|
| **TAM** | 总可及市场 |
| **CAGR** | 复合年均增长率 |
| **YoY** | 同比 |
| **MoM** | 环比 |

### 产品术语

| 术语 | 含义 |
|------|------|
| **ABF** | 味之素增层薄膜 |
| **BT** | BT 树脂 |
| **FCBGA** | 倒装芯片球栅阵列 |
| **FCCSP** | 倒装芯片级封装 |
| **HDI** | 高密度互连 |
| **CoWoS** | 基板上芯片 |

### 分析术语

| 术语 | 含义 |
|------|------|
| **FCF** | 自由现金流 |
| **WACC** | 加权平均资本成本 |
| **Ke** | 股权成本 |
| **TV** | 终值 |
| **P/E** | 市盈率 |
| **EV/EBITDA** | 企业价值倍数 |
