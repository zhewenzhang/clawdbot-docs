## 二、DCF 关键假设

| 参数 | 数值 | 说明 |
|------|------|------|
| **WACC** | 9.5% | 加权平均资本成本 |
| **永续增长率 (g)** | 2.5% | 长期增长假设 |
| **预测期** | 5 年 | 详细预测年份 |
| **FCF/净利润** | 60% | 自由现金流转换率 |

### WACC 计算参考

```
WACC = (E/V) × Ke + (D/V) × Kd × (1-T)

假设：
- 股权占比 (E/V)：90%
- 债务占比 (D/V)：10%
- 股权成本 (Ke)：10%（CAPM 模型估算）
- 债务成本 (Kd)：4%
- 税率 (T)：20%

WACC = 90% × 10% + 10% × 4% × (1-20%) = 9.5%
```

### 计算说明

- **现值系数**：1 / (1+WACC)^n
- **现值**：FCF × 现值系数

## 四、终值计算

### 方法1：永续增长法

```
TV = FCF₅ × (1 + g) / (WACC - g)
   = $38.9B × (1 + 2.5%) / (9.5% - 2.5%)
   = $38.9B × 1.025 / 0.07
   = $570.0B
```

## 八、估值模型速查表

### 核心公式

```
1. FCF = 净利润 + D&A - Capex - 营运资本增加

2. PV = FCF / (1+WACC)^n

3. TV = FCFₙ × (1+g) / (WACC-g)

4. DCF = Σ PV + TV现值

5. WACC = (E/V)×Ke + (D/V)×Kd×(1-T)
```

| 参数 | 保守 | 中性 | 乐观 |
