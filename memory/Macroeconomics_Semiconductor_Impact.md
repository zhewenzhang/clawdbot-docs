## 二、利率对半导体行业影响

### 2.1 利率传导机制

```
利率变化 → 半导体行业影响

    利率上升
        │
        ├──→ 融资成本上升
        │       │
        │       ├──→ Capex 减少
        │       ├──→ 扩产放缓
        │       └──→ 设备投资延迟
        │
        ├──→ 估值压缩
        │       │
        │       ├──→ P/E 下降
        │       └──→ 股价下跌
        │
        └──→ 需求抑制
                │
                ├──→ 终端需求下降
                └──→ 库存增加

    利率下降
        ├──→ 融资成本下降
        │       │
        │       ├──→ Capex 增加
        │       ├──→ 扩产加速
        │       └──→ 设备投资增加
        │
        ├──→ 估值扩张
        │       │
        │       ├──→ P/E 上升
        │       └──→ 股价上涨
        │
        └──→ 需求刺激
                │
                ├──→ 终端需求回升
                └──→ 库存去化
```

### 2.2 历史利率与半导体表现

| 时期 | 利率变化 | 半导体表现 | 案例 |
|------|----------|------------|------|
| 2020-2021 | 降息 | 大涨 | 疫情后需求爆发 |
| 2022 | 加息 | 大跌 | 美联储激进加息 |
| 2023 H2 | 加息见顶 | 反弹 | 预期降息 |
| 2024 | 降息预期 | 上涨 | AI 驱动 |

### 2.3 利率敏感性分析

| 公司类型 | 利率敏感性 | 原因 |
|----------|------------|------|
| **设计公司** | 高 | 高增长预期，估值敏感 |
| **设备公司** | 中 | Capex 投入与利率相关 |
| **代工/封测** | 低 | 相对稳定，重资产 |
| **存储芯片** | 高 | 周期性强，估值波动大 |

### 2.4 利率情景分析

| 情景 | 美联储利率 | 对半导体影响 |
|------|------------|--------------|
| **乐观** | 降息至 3% | 估值扩张，Capex 增加 |
| **基准** | 维持在 5% | 中性，影响有限 |
| **悲观** | 加息至 6% | 估值压缩，Capex 减少 |

## 三、汇率对半导体行业影响

### 3.1 汇率传导机制

```
汇率变化 → 半导体行业影响

    人民币升值
        │
        ├──→ 进口成本下降
        │       │
        │       ├──→ 设备采购成本降低
        │       └──→ 材料成本降低
        │
        ├──→ 出口竞争力下降
        │       │
        │       ├──→ 海外收入减少
        │       └──→ 毛利率承压
        │
        └──→ 报表影响
                │
                ├──→ 美元计价收入折算减少
                └──→ 海外子公司利润折算减少

    人民币贬值
        ├──→ 进口成本上升
        │       ├──→ 设备采购成本上升
        │       └──→ 材料成本上升
        │
        ├──→ 出口竞争力提升
        │       ├──→ 海外收入增加
        │       └──→ 毛利率改善
        │
        └──→ 报表影响
                ├──→ 美元计价收入折算增加
                └──→ 海外子公司利润折算增加
```

### 3.2 汇率敏感性分析

| 公司类型 | 汇率敏感性 | 影响方向 |
|----------|------------|----------|
| **出口导向** | 高 | 贬值受益 |
| **进口导向** | 高 | 升值受益 |
| **全球化公司** | 中 | 双刃剑 |
| **区域性公司** | 低 | 影响有限 |

### 3.3 主要汇率影响分析

| 汇率对 | 影响公司 | 影响程度 |
|--------|----------|----------|
| **USD/CNY** | 全部 | 高 |
| **USD/TWD** | 台积电 | 高 |
| **USD/JPY** | 日本设备/材料 | 中 |
| **USD/KRW** | 三星/SK 海力士 | 中 |

### 3.4 汇率风险管理

| 方法 | 说明 | 适用场景 |
|------|------|----------|
| **自然对冲** | 收入和成本同币种 | 全球化公司 |
| **远期合约** | 锁定汇率 | 短期汇率风险 |
| **期权合约** | 汇率保险 | 汇率波动大 |
| **区域多元化** | 分散汇率风险 | 长期策略 |

## 八、学习要点总结

### 8.1 核心概念

| 概念 | 含义 | 投资意义 |
|------|------|----------|
| **经济周期** | 复苏-扩张-滞胀-衰退 | 决定配置方向 |
| **利率传导** | 利率→成本/估值/需求 | 判断行业表现 |
| **汇率影响** | 进出口/报表/竞争力 | 调整配置结构 |
| **通胀效应** | 成本/需求/政策 | 识别风险机会 |

### 8.2 关键指标

| 指标 | 来源 | 预测价值 |
|------|------|----------|
| **PMI** | 统计局 | 需求先行 |
| **CPI/PPI** | 统计局 | 通胀水平 |
| **利率决议** | 美联储/央行 | 政策方向 |
| **GDP** | 统计局 | 经济趋势 |

### 8.3 投资原则

1. **关注宏观周期**：在复苏期/扩张期增持
2. **利率敏感度**：不同行业对利率敏感度不同
3. **汇率风险管理**：全球化公司需对冲
4. **通胀应对**：高通胀时关注资源股
5. **领先指标**：用 PMI、库存等领先指标预判
