## 一、投资决策框架

### 1.1 五维评估模型

| 维度 | 评估问题 | 权重 |
|------|----------|------|
| **行业** | 行业增速？竞争格局？ | 20% |
| **公司** | 竞争优势？增长动能？ | 30% |
| **估值** | 估值合理？ | 25% |
| **时机** | 周期位置？催化剂？ | 15% |
| **风险** | 风险可控？ | 10% |

### 1.2 投资决策流程

```
Step 1: 行业筛选
   - 行业增速 > 10%？
   - 竞争格局稳定？

Step 2: 公司筛选
   - 市场份额前三？
   - 竞争优势明显？

Step 3: 估值分析
   - P/E 低于历史均值？
   - DCF 内在价值 > 市值？

Step 4: 时机判断
   - 处于周期底部/上升期？
   - 有催化剂？

Step 5: 风险评估
   - 最大下跌空间？
   - 风险收益比 > 2:1？
```

## 二、行业分析检查

## 四、估值分析检查

| 检查项 | 核查要点 |
|--------|----------|
| **P/E 比较** | 比同业贵还是便宜？ |

| 检查项 | 核查要点 |
|--------|----------|
| **假设合理性** | 营收增长假设是否合理？ |
| **WACC** | WACC 是否合理？ |
| **永续增长** | 永续增长率是否合理？ |
| **估值结果** | DCF 内在价值 > 市值？ |

### 4.4 估值检查清单

| 检查项 | 是 | 否 |
|--------|----|----|
| P/E < 历史 30 分位？ | ☐ | ☐ |
| P/E < 同业中位数？ | ☐ | ☐ |
| PEG < 1？ | ☐ | ☐ |
| DCF 隐含上涨空间 > 20%？ | ☐ | ☐ |
| 股息率 > 2%？ | ☐ | ☐ |

## 十、学习要点总结

### 10.1 投资决策框架

```
行业 → 公司 → 估值 → 时机 → 风险
```

### 10.2 核心检查项

| 维度 | 核心问题 |
|------|----------|
| **行业** | 行业增速？竞争格局？ |
| **公司** | 竞争优势？增长动能？ |
| **估值** | 估值合理？ |
| **时机** | 周期位置？ |
| **风险** | 风险可控？ |

### 10.3 投资建议原则

1. **只投资懂的**：不懂不投
2. **安全边际**：低估值时买入
3. **分散投资**：不要 all-in 单只
4. **动态调整**：定期复盘
5. **止损纪律**：设止损位
