## 一、资金流向基础概念

### 1.1 什么是资金流向

资金流向反映的是股票交易中资金的净流入或净流出情况，是判断市场情绪和主力动向的重要指标。

### 1.2 资金流向分类

| 类型 | 金额 | 含义 |
|------|------|------|
| **主力资金** | >100万/笔 | 机构、大户 |
| **中单资金** | 10-100万/笔 | 中型投资者 |
| **散户资金** | <10万/笔 | 个人投资者 |

### 1.3 关键指标

| 指标 | 计算 | 含义 |
|------|------|------|
| **主力净流入** | 主力买入 - 主力卖出 | 机构资金动向 |
| **散户净流入** | 散户买入 - 散户卖出 | 散户情绪 |
| **净流入率** | 净流入 / 成交额 | 资金集中度 |
| **资金流向 (N日)** | N日累计净流入 | 趋势判断 |

## 二、资金流向分析框架

### 2.1 分析步骤

```
Step 1: 获取数据
   - 当日主力/散户净流入
   - 5日/10日/20日累计净流入
   - 成交额和成交量

Step 2: 判断方向
   - 主力净流入为正 → 机构看好
   - 主力净流入为负 → 机构看空
   - 散户净流入为正 → 散户追涨

Step 3: 评估持续性
   - 5日趋势：短期动向
   - 10日趋势：中期趋势
   - 20日趋势：长期趋势

Step 4: 验证基本面
   - 资金流入是否配合基本面改善？
   - 是否只是题材炒作？

Step 5: 做出判断
   - 资金+基本面+技术面共振 → 买入信号
   - 资金流出+基本面恶化 → 卖出信号
```

### 2.2 资金流向信号

| 主力资金 | 散户资金 | 信号强度 | 操作建议 |
|----------|----------|----------|----------|
| **净流入** | 净流出 | ⭐⭐⭐⭐⭐ 强 | 机构建仓，积极关注 |
| **净流入** | 净流入 | ⭐⭐⭐ 中 | 可能有行情，谨慎参与 |
| **净流出** | 净流入 | ⭐⭐ 弱 | 机构出货，远离 |
| **净流出** | 净流出 | ⭐ 最弱 | 恐慌抛售，观望 |

## 四、资金流选股策略

### 4.1 强势股筛选

```
条件：
1. 5日主力净流入 > 1亿元
2. 5日涨幅 > 10%
3. 成交额 > 5亿元（流动性）
4. 主力净流入占比 > 30%

筛选出：资金+涨幅共振的强势股
```

### 4.2 机构建仓识别

```
条件：
1. 20日主力净流入 > 5亿元
2. 20日涨幅 > 20%
3. 成交量放大（较20日均量 +50%）
4. 股价回踩 20日均线获得支撑

识别：机构正在建仓的股票
```

### 4.3 主力出货识别

```
条件：
1. 5日主力净流出 > 1亿元
2. 5日跌幅 > 10%
3. 成交量放大（恐慌抛售）
4. 散户净流入为正（接盘）

识别：主力正在出货的股票
```

## 九、学习总结

### 核心概念

1. **资金流向反映市场情绪**：主力资金代表机构判断
2. **资金流需要结合基本面**：避免题材炒作陷阱
3. **持续性比单日更重要**：趋势形成需要时间
4. **半导体行业有周期性**：资金流与周期密切相关

### 分析框架

```
获取数据 → 判断方向 → 评估持续性 → 验证基本面 → 做出决策
```

### 关键指标

- 主力净流入/流出
- 5日/10日/20日累计净流入
- 净流入占比
- 北上资金动向

### 注意事项

1. 不要单独依赖资金流
2. 关注持续性而非单日数据
3. 结合基本面验证
4. 设置止损，控制风险
