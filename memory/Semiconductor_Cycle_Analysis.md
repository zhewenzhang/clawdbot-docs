## 一、半导体周期概述

### 1.1 多重周期叠加

```
半导体周期 = 硅周期 + 库存周期 + 产能周期 + 创新周期

各周期特征：
├── 硅周期（3-4年）：需求端波动
├── 库存周期（2-3年）：渠道库存调整
├── 产能周期（5-7年）：供给端扩张
└── 创新周期（10年+）：技术革命驱动
```

### 1.2 周期传导机制

```
创新周期（新产品）→ 硅周期（需求增长）
    ↓
产能周期（供给扩张）
    ↓
库存周期（渠道库存调整）
    ↓
价格周期（供需失衡）
    ↓
再次循环
```

## 三、库存周期分析

### 3.1 库存周期阶段

```
阶段 1：主动补库存 → 需求增长，库存增加
阶段 2：被动补库存 → 需求放缓，库存继续增加
阶段 3：主动去库存 → 需求下降，库存减少
阶段 4：被动去库存 → 需求回升，库存减少
```

### 3.2 库存周期指标

| 指标 | 计算 | 健康范围 |
|------|------|----------|
| **库存天数** | 存货 / COGS × 365 | 60-90 天 |
| **库存周转率** | COGS / 存货 | 4-6 次/年 |
| **库存/营收比** | 存货 / 营收 | 15-25% |

### 3.3 半导体库存周期（2023-2024）

| 时间 | 库存天数 | 趋势 | 阶段 |
|------|----------|------|------|
| 2023 Q1 | 90+ 天 | 上升 | 被动补库存 |
| 2023 Q3 | 100 天 | 峰值 | 被动补库存 |
| 2023 Q4 | 90 天 | 下降 | 主动去库存 |
| 2024 Q1 | 75 天 | 下降 | 主动去库存 |
| 2024 Q2 | 65 天 | 下降 | 被动去库存 |

## 九、学习要点总结

### 9.1 周期分析框架

```
1. 识别周期类型（硅/库存/产能/创新）
2. 判断周期位置（底部/顶部/中间）
3. 预测周期方向（向上/向下/震荡）
4. 选择投资策略（配置/持有/减仓）
```

### 9.2 核心指标

| 指标 | 底部信号 | 顶部信号 |
|------|----------|----------|
| **营收增速** | 转正 | 触顶 |
| **库存天数** | <60 天 | >90 天 |
| **产能利用率** | <70% | >90% |
| **P/E** | <15x | >30x |

### 9.3 投资原则

1. **周期底部布局**：估值低、预期低
2. **周期顶部减仓**：估值高、预期高
3. **关注创新周期**：新技术带来新需求
4. **分散配置**：不同周期位置的公司
