## 一、DCF 估值模型

### 1.1 DCF 核心公式

```
企业价值 = Σ [FCFₜ / (1+WACC)ᵗ] + TV / (1+WACC)ⁿ

其中：
- FCF = 自由现金流
- WACC = 加权平均资本成本
- TV = 终值（Terminal Value）
- t = 年份
- n = 预测期
```

### 1.2 自由现金流 (FCF) 计算

```
FCF = EBIT × (1 - 税率) + D&A - 资本支出 - 营运资本变动

简化版：
FCF = 净利润 + 折旧摊销 - 资本支出 - 营运资本增加
```

### 1.3 终值 (TV) 计算

**永续增长法**：
```
TV = FCFₙ × (1 + g) / (WACC - g)

其中 g = 永续增长率（通常 2-3%）
```

**退出倍数法**：
```
TV = EBITDAₙ × 退出倍数
```

### 1.4 WACC 计算

```
WACC = (E/V) × Ke + (D/V) × Kd × (1 - 税率)

其中：
- E = 股权市值
- D = 债务市值
- V = 企业价值 (E+D)
- Ke = 股权成本（CAPM 模型）
- Kd = 债务成本
```

### 1.5 CAPM 模型（股权成本）

```
Ke = Rf + β × (Rm - Rf) + ERP

其中：
- Rf = 无风险利率（通常用 10 年期国债）
- β = 贝塔系数（相对于大盘的系统性风险）
- Rm = 市场预期收益率
- ERP = 股权风险溢价 (Rm - Rf)
```

## 二、半导体行业估值倍数

### 2.1 常用估值指标

| 指标 | 计算公式 | 含义 |
|------|----------|------|
| **P/E** | 市值 / 净利润 | 市盈率，成长股常用 |
| **EV/EBITDA** | 企业价值 / EBITDA | 剔除非现金影响 |
| **P/S** | 市值 / 营收 | 营收驱动的估值 |
| **P/B** | 市值 / 账面价值 | 资产密集型行业 |
| **EV/Revenue** | 企业价值 / 营收 | SaaS/平台公司 |

### 2.2 半导体行业典型估值

| 公司类型 | P/E 范围 | EV/EBITDA 范围 | 特点 |
|----------|-----------|----------------|------|
| **晶圆代工** | 15-25x | 8-12x | 资本密集，重资产 |
| **IC 设计** | 20-40x | 12-20x | 轻资产，高成长 |
| **封装测试** | 12-20x | 6-10x | 成熟行业 |
| **设备** | 15-30x | 10-15x | 周期性强 |
| **材料** | 20-35x | 12-18x | 国产替代 |

### 2.3 台积电估值案例

```
TSMC 估值参考（2024年数据）：

- 市值：约 $800B
- P/E：约 20-25x
- EV/EBITDA：约 10-12x
- 增长预期：年增 15-20%（AI 驱动）

DCF 关键假设：
- WACC：8-10%
- 永续增长率：2-3%
- 预测期：5 年
- 退出倍数：10-12x EBITDA
```

## 三、股票资金流向分析

### 3.1 资金流向指标

| 指标 | 计算 | 含义 |
|------|------|------|
| **主力净流入** | 大单净买入额 | 机构资金动向 |
| **散户净流入** | 小单净买入额 | 散户情绪 |
| **主力占比** | 主力买入 / 总成交 | 资金集中度 |
| **资金流向** | 5日/10日/20日净流入 | 趋势判断 |

### 3.2 行业资金轮动

**半导体行业资金特点**：

1. **周期性**：
   - 芯片景气周期：3-4 年
   - 上行周期：资金涌入
   - 下行周期：资金撤离

2. **主题驱动**：
   - AI 热潮：资金集中 AI 芯片
   - 国产替代：资金关注中国厂商
   - 汽车电子：资金布局车用芯片

3. **资金来源**：
   - 机构：公募基金、社保、外资
   - 游资：短线炒作
   - 北上资金：外资动向

### 3.3 资金流向分析框架

```
Step 1: 观察资金流向
- 主力净流入/流出？
- 持续性如何（5日/10日趋势）？

Step 2: 判断资金性质
- 机构买入：基本面驱动，长期持有
- 游资炒作：情绪驱动，短线操作

Step 3: 验证基本面
- 营收/利润增长？
- 估值合理性？
- 行业景气度？

Step 4: 做出决策
- 机构买入 + 基本面好 → 关注
- 游资炒作 + 基本面差 → 回避
```

## 四、关键财务指标速查

### 4.1 盈利能力指标

| 指标 | 计算 | 优秀 | 良好 | 一般 |
|------|------|------|------|------|
| **毛利率** | 营收 - 成本 / 营收 | >50% | 30-50% | <30% |
| **营业利润率** | 营业利润 / 营收 | >20% | 10-20% | <10% |
| **净利率** | 净利润 / 营收 | >15% | 8-15% | <8% |
| **ROE** | 净利润 / 股东权益 | >20% | 10-20% | <10% |
| **ROIC** | NOPAT / 投入资本 | >15% | 8-15% | <8% |

### 4.2 成长能力指标

| 指标 | 计算 | 含义 |
|------|------|------|
| **营收增速** | (本期营收 - 上期营收) / 上期营收 | 成长性 |
| **利润增速** | (本期利润 - 上期利润) / 上期利润 | 盈利成长 |
| **研发费用率** | 研发费用 / 营收 | 创新投入 |

### 4.3 运营效率指标

| 指标 | 计算 | 含义 |
|------|------|------|
| **资产周转率** | 营收 / 总资产 | 资产利用效率 |
| **存货周转率** | 营收 / 存货 | 库存管理 |
| **应收账款周转** | 营收 / 应收账款 | 回款效率 |

### 4.4 财务健康指标

| 指标 | 计算 | 健康范围 |
|------|------|----------|
| **负债率** | 总负债 / 总资产 | <50% |
| **流动比率** | 流动资产 / 流动负债 | >1.5 |
| **利息保障倍数** | EBIT / 利息费用 | >5x |

## 八、学习要点总结

### 核心公式

```
1. FCF = 净利润 + D&A - Capex - 营运资本增加
2. DCF = Σ FCF / (1+WACC)ᵗ + TV / (1+WACC)ⁿ
3. WACC = (E/V)×Ke + (D/V)×Kd×(1-T)
4. Ke = Rf + β×(Rm-Rf)
5. P/E = 市值 / 净利润
6. EV/EBITDA = 企业价值 / EBITDA
```

### 关键指标

| 类型 | 指标 |
|------|------|
| **成长** | 营收增速、利润增速 |
| **盈利** | 毛利率、净利率、ROE |
| **估值** | P/E、EV/EBITDA、DCF |
| **资金** | 主力净流入、北上资金 |

### 分析框架

1. **自上而下**：宏观经济 → 行业 → 个股
2. **自下而上**：个股基本面 → 估值 → 决策
3. **资金面验证**：资金流向确认趋势
