二、资金流向核心规律
===============================================================================

【规律1】资金轮动的三维驱动模型

资金流动并非随机，而是由估值、政策、风险三维因素共同驱动：

第一维度：估值驱动
- 底层逻辑：均值回归，资金从高估值流向低估值
- 当前表现：银行PB 0.5倍 vs 半导体PE 60倍，资金用脚投票
- 老板应用：理解自己关注的科技公司估值位置

第二维度：政策驱动
- 底层逻辑：事件驱动，资金提前1-2周埋伏政策受益行业
- 当前表现：地产刺激政策密集出台，工程机械、地产资金流入放大
- 老板应用：关注产业政策（半导体补贴、AI扶持等）

第三维度：风险偏好驱动
- 底层逻辑：风险偏好随经济周期变化，资金追求确定性vs弹性
- 当前表现：中等偏低风险偏好，资金偏好成熟行业
- 老板应用：判断市场风险偏好拐点，择时加仓科技

【规律2】行业轮动的周期规律

经济周期四阶段与行业轮动关系：

阶段1：复苏前期（当前）
- 轮动路径：金融 → 周期
- 驱动逻辑：经济预期改善，利率敏感型行业率先反弹
- 典型表现：银行、证券率先启动
- 老板策略：金融配置为主，科技观察

阶段2：复苏后期
- 轮动路径：周期 → 消费
- 驱动逻辑：实体经济复苏传导至消费需求
- 典型表现：白酒、空运开始获得资金关注
- 老板策略：逐步增加消费配置

阶段3：过热期
- 轮动路径：消费 → 科技
- 驱动逻辑：需求旺盛推动成长股估值扩张
- 典型表现：科技板块获得资金追捧
- 老板策略：加大科技配置，发挥专业优势

阶段4：衰退期
- 轮动路径：现金 → 科技
- 驱动逻辑：科技股防御属性凸显
- 老板策略：科技作为防御配置

【规律3】主力资金的运作模式

主力资金（超大单+大单）具有三大特征：

特征1：领先性
- 表现：主力资金流入领先股价1-2周
- 含义：散户看到股价上涨时，主力已建仓完毕
- 老板应用：关注资金流入信号，而非股价上涨后追高

特征2：持续性
- 表现：真正的资金流入需要连续性，而非单日脉冲
- 含义：单日大额流入可能是诱多，持续小额流入才是真流入
- 老板应用：观察5日、10日累计流入，而非单日数据

特征3：背离预警
- 表现：股价上涨但资金流出 = 主力出货
- 含义：散户追高接盘，主力高位派发
- 老板应用：避开资金流出+股价上涨的背离标的

三、资金流向选股方法论
===============================================================================

【方法论1】如何识别真正的资金流入

信号灯系统（绿灯行，红灯停）：

绿灯信号（买入确认）：
- 连续5日净流入 > 0
- 主力资金占比 > 40%（超大单+大单占比）
- 成交量较20日均量放大 > 20%
- 股价涨幅 < 行业平均5%（未充分反映资金流入）

黄灯信号（谨慎观察）：
- 3-4日连续流入，但主力占比30-40%
- 成交量放大10-20%
- 股价与行业平均持平

红灯信号（坚决回避）：
- 资金流出 > 2日
- 主力占比 < 30%
- 成交量萎缩
- 股价涨幅 > 行业平均10%

【方法论2】如何选择买入时机

时机1：行业资金首次突破0线
- 信号：行业5日累计资金由负转正
- 含义：资金开始关注该行业
- 操作：首次流入时建仓10%

时机2：个股资金首次绿翻红
- 信号：个股5日累计资金由负转正
- 含义：个股获得资金关注
- 操作：回调至20日均线附近买入

时机3：回踩支撑位
- 信号：股价回调至20日/60日均线
- 含义：资金承接有力
- 操作：在支撑位分批买入

时机4：突破整理平台
- 信号：股价放量突破整理区间上沿
- 含义：资金推动上涨
- 操作：突破后回踩确认时买入

【方法论3】如何设定止损位

止损类型1：核心逻辑止损
- 触发条件：资金流向由流入转流出（5日累计转负）
- 含义：资金不再看好，逻辑破坏
- 操作：立即止损

止损类型2：技术止损
- 触发条件：亏损达到10%
- 含义：股价走势弱于预期
- 操作：严格执行止损

止损类型3：移动止盈
- 触发条件：盈利50%后提高止盈线至30%
- 含义：保护盈利，避免坐电梯
- 操作：分批止盈

- 能获取：主要客户名单、客户占比
- 应用于：评估收入质量
- 示例：头部客户占比 > 50% = 客户质量高

半导体选股要点（老板专用）：
1. 优先选择：产能利用率回升、订单真实性高、技术节点领先的公司
