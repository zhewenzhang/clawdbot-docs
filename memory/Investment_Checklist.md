## 一、五维评估模型

### 1.1 模型概述

```
五维评估模型

┌─────────────────────────────────────────────────────────┐
│                   投资评估五维度                         │
├───────────┬───────────┬───────────┬─────────┬───────────┤
│   行业    │   公司    │   财务    │  估值   │   风险    │
│   维度    │   维度    │   维度    │  维度   │   维度    │
├───────────┼───────────┼───────────┼─────────┼───────────┤
│ 行业规模  │ 竞争优势  │ 盈利能力  │ DCF 估值│ 周期位置  │
│ 增长性    │ 管理层    │ 成长性    │ 相对估值│ 风险因素  │
│ 竞争格局  │ 技术实力  │ 现金流    │ 估值区间│ 催化剂    │
│ 政策环境  │ 客户结构  │ 资产负债  │ 溢价/折价│ 对冲策略 │
└───────────┴───────────┴───────────┴─────────┴───────────┘
```

### 1.2 评分标准

| 维度 | 权重 | 评分（1-5） | 说明 |
|------|------|-------------|------|
| **行业** | 25% | 1-5 | 行业吸引力 |
| **公司** | 25% | 1-5 | 公司竞争力 |
| **财务** | 20% | 1-5 | 财务健康度 |
| **估值** | 20% | 1-5 | 估值吸引力 |
| **风险** | 10% | 1-5 | 风险可控度 |

### 1.3 综合评分计算

```
综合评分 = Σ（维度评分 × 维度权重）

评分解释：
- 4.0-5.0：优秀，重仓
- 3.5-4.0：良好，标配
- 3.0-3.5：一般，低配
- 2.5-3.0：较差，观望
- <2.5：很差，回避
```

## 五、估值维度评估

### 5.1 DCF 估值评估

| 评估项 | 计算方法 | 评分标准 |
|--------|----------|----------|
| **DCF 内在价值** | 模型计算 | 当前股价 < DCF×0.8(5), <DCF(4), >DCF×1.2(3) |
| **安全边际** | (DCF-股价)/DCF | >30%(5), 20-30%(4), 10-20%(3), <10%(2) |

### 5.2 相对估值评估

| 评估项 | 计算方法 | 评分标准 |
|--------|----------|----------|
| **P/E** | 当前 P/E vs 行业 | 低于 20%(5), 低于 10%(4), 持平(3), 高于(2) |
| **EV/EBITDA** | 当前 EV/EBITDA vs 行业 | 低于 20%(5), 低于 10%(4), 持平(3), 高于(2) |
| **PEG** | PEG 值 | <0.8(5), 0.8-1.0(4), 1.0-1.2(3), >1.2(2) |

## 九、学习要点总结

### 9.1 五维评估模型

| 维度 | 权重 | 核心指标 |
|------|------|----------|
| **行业** | 25% | 规模、增长、竞争格局 |
| **公司** | 25% | 竞争优势、管理层、技术 |
| **财务** | 20% | 盈利、成长、健康 |
| **估值** | 20% | DCF、相对估值、安全边际 |
| **风险** | 10% | 周期位置、风险因素、催化剂 |

### 9.2 投资决策流程

1. **行业分析**：评估行业吸引力
2. **公司分析**：评估公司竞争力
3. **财务分析**：评估财务健康度
4. **估值分析**：评估估值合理性
5. **风险分析**：评估风险可控度
6. **综合评分**：计算综合评分
7. **投资决策**：给出投资建议
8. **跟踪监控**：持续跟踪更新

### 9.3 关键要点

- **系统性**：避免主观臆断
- **可量化**：评分标准化
- **可追踪**：定期更新评分
- **灵活性**：根据情况调整
