## 二、股票筛选框架

### 2.1 成长股筛选

```
条件（成长股）：
1. 营收增速 > 20%
2. 毛利率 > 40%
3. ROE > 15%
4. 研发费用率 > 15%
5. AI/数据中心相关收入占比 > 30%

筛选结果：NVIDIA、台积电、AMD
```

### 2.2 价值股筛选

```
条件（价值股）：
1. P/E < 15x
2. P/B < 3x
3. 股息率 > 2%
4. 自由现金流为正
5. 市场份额稳定

筛选结果：英特尔、德州仪器、联电
```

### 2.3 周期股筛选

```
条件（周期股）：
1. 行业周期位置（底部/顶部）
2. 产能利用率
3. 库存天数
4. 资本支出计划
5. 客户结构

关注指标：
- 产能利用率 > 90%：供需紧张
- 库存天数 < 60 天：健康
- Capex 增速放缓：周期见顶
```
