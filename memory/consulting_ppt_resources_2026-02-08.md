## 一、MBB三大咨询公司核心方法论

### 1. McKinsey (麦肯锡)

#### 核心原则
- **金字塔原理 (Minto Pyramid Principle)**: 结论先行，以上统下
- **MECE原则**: 相互独立，完全穷尽 (Mutually Exclusive, Collectively Exhaustive)
- **假设驱动 (Hypothesis-Driven)**: 先建立假设，再验证

#### 常用框架
| 框架名称 | 适用场景 |
|---------|---------|
| 3C模型 | 市场分析 |
| 7S模型 | 组织诊断 |
| 价值链分析 | 竞争优势 |
| 波特五力 | 行业分析 |

#### 报告特点
- 简约设计，数据驱动
- 结论先行，每页都有"电梯演讲"
- 图表优先，文字精炼

### 2. BCG (波士顿咨询)

#### 核心工具
| 工具名称 | 说明 |
|---------|------|
| **BCG矩阵** | 现金牛、明星、问号、狗 |
| **增长-份额矩阵** | 市场增长率 vs 市场份额 |
| **经验曲线** | 成本随经验下降 |
| **GE矩阵** | 行业吸引力 vs 竞争力 |

#### 分析方法
- 问题定义 → 事实收集 → 假设验证 → 建议提出
- 结构化思考，层层递进

### 3. Bain (贝恩)

#### 核心方法
- **Bain Value Creator**: 价值创造分析
- **MECE原则**: 与McKinsey类似
- **数字驱动**: 重视数据验证

#### 报告风格
- 逻辑严密，证据充分
- 可执行建议，量化影响

## 三、咨询报告设计最佳实践

### 1. 结构设计
```
【标准咨询报告结构】
├── 执行摘要 (Executive Summary)
│   └── 结论先行，1页纸
├── 问题定义 (Problem Statement)
│   └── 明确范围和目标
├── 分析框架 (Framework)
│   └── 选择合适模型
├── 数据分析 (Analysis)
│   └── 事实驱动，MECE
├── 洞察发现 (Insights)
│   └── 假设验证
└── 建议行动 (Recommendations)
    └── 可执行方案
```

### 2. 排版原则
| 原则 | 说明 |
|-----|------|
| **KISS原则** | Keep It Simple, Stupid |
| **一页一观点** | 每页只讲一个核心 |
| **图表优先** | 用图不用表，用表不用字 |
| **留白** | 30%空白，视觉呼吸 |
| **字体层级** | 标题→小标题→正文 |

### 3. 配色方案
```
【经典咨询配色】
├── 主色: 深蓝 #003366 (McKinsey风格)
├── 强调色: 金色 #C4A962 (高端感)
├── 辅助色: 灰色 #666666 (平衡)
└── 背景: 白色 #FFFFFF (简洁)
```

### 4. 字体规范
```
【推荐字体】
├── 标题: 18-24px (粗体)
├── 小标题: 14-16px (中等)
├── 正文: 11-12px (易读)
└── 注释: 9-10px (辅助)
```
