## 二、情景分析方法

### 2.1 情景分析原理

**情景分析 = 设定多个假设情景，评估结果**

```
情景分析结构

┌─────────────────────────────────────────────────────┐
│                     基础情景                         │
│                   （基准假设）                       │
│                   概率：50%                         │
│                   结果：$3.0                        │
├─────────────────────────────────────────────────────┤
│                                                     │
│  ┌─────────────┐      ┌─────────────┐              │
│  │   乐观情景   │      │   悲观情景   │              │
│  │  概率：25%   │      │  概率：25%   │              │
│  │  结果：$4.0  │      │  结果：$2.0  │              │
│  └─────────────┘      └─────────────┘              │
│                                                     │
└─────────────────────────────────────────────────────┘
```

### 2.2 情景设定

| 情景 | 概率 | 营收假设 | 毛利率假设 | 费用率假设 |
|------|------|----------|------------|------------|
| **乐观** | 25% | +30% | 28% | 10% |
| **基准** | 50% | +20% | 25% | 11% |
| **悲观** | 25% | +10% | 22% | 12% |

### 2.3 情景分析示例

**案例：某 IC 载板公司**

**1. 营收假设**

| 情景 | 2024 营收 | 增速 | 备注 |
|------|-----------|------|------|
| **乐观** | $1,224M | +30% | AI 需求爆发 |
| **基准** | $1,020M | +20% | 正常增长 |
| **悲观** | $816M | +10% | 需求放缓 |

**2. 毛利率假设**

| 情景 | 毛利率 | 备注 |
|------|--------|------|
| **乐观** | 28% | 高端产品占比提升 |
| **基准** | 25% | 正常水平 |
| **悲观** | 22% | 产能过剩，价格下降 |

**3. 净利润计算**

| 情景 | 营收 | 毛利率 | 毛利 | 费用 | 净利 | 净利率 |
|------|------|--------|------|------|------|--------|
| **乐观** | $1,224M | 28% | $343M | $122M | $166M | 13.6% |
| **基准** | $1,020M | 25% | $255M | $112M | $107M | 10.5% |
| **悲观** | $816M | 22% | $180M | $98M | $61M | 7.5% |

**4. 估值计算（DCF）**

| 情景 | 净利 | EPS | P/E | 股价 |
|------|------|-----|-----|------|
| **乐观** | $166M | $0.17 | 25x | $4.25 |
| **基准** | $107M | $0.11 | 20x | $2.20 |
| **悲观** | $61M | $0.06 | 15x | $0.90 |

**5. 加权估值**

```
加权估值 = Σ（情景概率 × 情景估值）

= 25% × $4.25 + 50% × $2.20 + 25% × $0.90
= $1.06 + $1.10 + $0.23
= $2.39
```

### 2.4 情景分析应用

| 应用场景 | 情景设定 |
|----------|----------|
| **宏观经济** | 复苏/中性/衰退 |
| **行业周期** | 上行/中性/下行 |
| **公司经营** | 乐观/基准/悲观 |
| **政策变化** | 利好/中性/利空 |

## 三、敏感性分析方法

### 3.1 敏感性分析原理

**敏感性分析 = 改变单一变量，观察结果变化**

```
敏感性分析结构

      变量 A 变动    →    结果变动
      变量 B 变动    →    结果变动
      变量 C 变动    →    结果变动
      
      敏感度 = 结果变动 / 变量变动
```

### 3.2 敏感性分析示例

**案例：某 IC 载板公司 DCF 估值**

**1. 关键变量**

| 变量 | 基准值 | 变动幅度 | 敏感度 |
|------|--------|----------|--------|
| **营收增速** | +20% | ±5% | 高 |
| **毛利率** | 25% | ±3% | 高 |
| **WACC** | 9% | ±1% | 中 |
| **永续增长率** | 3% | ±1% | 低 |

**2. 敏感性矩阵**

**营收增速敏感性**

| 营收增速 | 估值 | 变动 |
|----------|------|------|
| +15% | $2.00 | -17% |
| +20%（基准） | $2.40 | 0% |
| +25% | $2.85 | +19% |

**WACC 敏感性**

| WACC | 估值 | 变动 |
|------|------|------|
| 8.0% | $2.70 | +13% |
| 9.0%（基准） | $2.40 | 0% |
| 10.0% | $2.15 | -10% |

**3. 敏感性图表**

```
营收增速敏感性              WACC 敏感性

$3.00 ─┤●                    $3.00 ─┤
       │  ●                   │     ●
$2.40 ─┤  ●基准              $2.40 ─┤  ●基准
       │  ●                   │     ●
$1.80 ─┤  ●                  $1.80 ─┤
       └────────→              └────────→
        15% 20% 25%             8% 9% 10%
```

### 3.3 敏感性分析应用

| 变量 | 敏感度 | 监控重点 |
|------|--------|----------|
| **营收增速** | 高 | 市场变化、份额变化 |
| **毛利率** | 高 | 价格、成本结构 |
| **WACC** | 中 | 利率、Beta |
| **Capex** | 中 | 投资计划 |
| **永续增长率** | 低 | 长期增长预期 |

## 四、VaR（风险价值）方法

### 4.1 VaR 原理

**VaR = 在给定置信水平下，可能的最大损失**

```
VaR 定义

置信水平：95%
持有期：1 天
VaR = $100万

含义：
- 明天有 95% 的概率，损失不会超过 $100万
- 或者说，有 5% 的概率，损失会超过 $100万
```

### 4.2 VaR 计算方法

| 方法 | 原理 | 优点 | 缺点 |
|------|------|------|------|
| **历史模拟法** | 用历史数据模拟 | 简单 | 依赖历史 |
| **方差-协方差法** | 正态分布假设 | 快速 | 假设局限 |
| **Monte Carlo** | 随机模拟 | 灵活 | 计算复杂 |

### 4.3 VaR 计算示例

**案例：半导体组合**

**1. 组合数据**

| 股票 | 权重 | 日波动率 |
|------|------|----------|
| **台积电** | 40% | 2.5% |
| **NVIDIA** | 30% | 3.5% |
| **AMD** | 20% | 4.0% |
| **中芯国际** | 10% | 5.0% |

**2. 计算组合 VaR（95%，1 天）**

假设：
- 日波动率（组合）：σp = 2.8%
- 组合价值：$10M
- Z 值（95%）：1.65

```
VaR = 组合价值 × σp × Z
    = $10M × 2.8% × 1.65
    = $462,000

含义：
- 明天有 95% 的概率，损失不会超过 $46.2万
- 或：有 5% 的概率，损失会超过 $46.2万
```

**3. VaR 扩展到更长时间**

| 持有期 | 放大系数 | VaR |
|--------|----------|-----|
| 1 天 | 1.0x | $0.46M |
| 1 周 | √7 = 2.65x | $1.22M |
| 1 月 | √22 = 4.69x | $2.16M |
| 1 年 | √252 = 15.87x | $7.30M |

### 4.4 VaR 应用

| 应用场景 | 说明 |
|----------|------|
| **风险限额** | 设置 VaR 限额 |
| **绩效归因** | 风险调整收益 |
| **资本配置** | 风险资本分配 |
| **压力测试** | 极端情况测试 |

## 五、Monte Carlo 模拟方法

### 5.1 Monte Carlo 原理

**Monte Carlo = 随机抽样，大量模拟**

```
Monte Carlo 模拟

┌─────────────────────────────────────────┐
│           设定参数分布                   │
│         正态分布、假设均值/标准差        │
└──────────────────┬──────────────────────┘
                   ▼
┌─────────────────────────────────────────┐
│           随机抽样                       │
│         生成随机数，模拟参数值           │
└──────────────────┬──────────────────────┘
                   ▼
┌─────────────────────────────────────────┐
│           计算结果                       │
│         根据参数计算估值                 │
└──────────────────┬──────────────────────┘
                   ▼
┌─────────────────────────────────────────┐
│           重复模拟                       │
│         重复 10,000 次                  │
└──────────────────┬──────────────────────┘
                   ▼
┌─────────────────────────────────────────┐
│           结果分析                       │
│         估值分布、概率、VaR              │
└─────────────────────────────────────────┘
```

### 5.2 Monte Carlo 示例

**案例：某 IC 载板公司估值**

**1. 参数分布假设**

| 参数 | 分布 | 均值 | 标准差 |
|------|------|------|--------|
| **营收增速** | 正态 | 20% | 5% |
| **毛利率** | 正态 | 25% | 2% |
| **WACC** | 正态 | 9% | 0.5% |
| **永续增长率** | 正态 | 3% | 0.5% |

**2. 模拟结果（10,000 次）**

```
Monte Carlo 估值分布

频率
  ↑
  │          ┌─┐
  │        ┌─┘ └─┐
  │      ┌─┘     └─┐
  │    ┌─┘         └─┐
  │  ┌─┘             └─┐
  │┌─┘                 └─┐
  ││                     │
  └──│────────────────────│──→
   $1.0  $2.0  $2.5  $3.0  $4.0

统计结果：
- 平均估值：$2.50
- 中位数：$2.45
- 标准差：$0.50
- 最低5%分位：$1.70
- 最高95%分位：$3.50
```

**3. 概率分析**

| 估值区间 | 概率 |
|----------|------|
| < $1.5 | 5% |
| $1.5 - $2.0 | 15% |
| $2.0 - $2.5 | 35% |
| $2.5 - $3.0 | 30% |
| $3.0 - $3.5 | 10% |
| > $3.5 | 5% |

### 5.3 Monte Carlo 优势

| 优势 | 说明 |
|------|------|
| **灵活性** | 可处理复杂分布 |
| **全面性** | 考虑变量相关性 |
| **概率化** | 结果是概率分布 |
| **敏感性** | 自然包含敏感性 |

## 六、压力测试方法

### 6.1 压力测试原理

**压力测试 = 极端情景下的测试**

```
压力测试 vs VaR

VaR：
- 正常情况
- 95% 置信水平
- 最大损失：$100万

压力测试：
- 极端情况
- 如：市场下跌 30%
- 损失：$300万
```

### 6.2 压力测试情景

| 情景 | 设定 | 影响 |
|------|------|------|
| **市场崩溃** | 标普 500 下跌 30% | 估值下跌 40% |
| **行业衰退** | 半导体下滑 20% | 营收下降 25% |
| **公司危机** | 主要客户流失 | 营收下降 30% |
| **利率飙升** | 利率上升 3% | WACC 上升 1% |

### 6.3 压力测试示例

**案例：某半导体组合**

**压力情景**

| 情景 | 市场变化 | 组合变化 |
|------|----------|----------|
| **轻度压力** | -10% | -12% |
| **中度压力** | -20% | -25% |
| **重度压力** | -30% | -38% |
| **极端压力** | -40% | -50% |

**压力测试结果**

| 情景 | 组合价值 | 损失 | 损失率 |
|------|----------|------|--------|
| **正常** | $10M | $0 | 0% |
| **轻度压力** | $8.8M | -$1.2M | -12% |
| **中度压力** | $7.5M | -$2.5M | -25% |
| **重度压力** | $6.2M | -$3.8M | -38% |
| **极端压力** | $5.0M | -$5.0M | -50% |
