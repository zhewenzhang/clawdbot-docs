## 四、终值计算

### 永续增长法

```
TV = FCF₅ × (1 + g) / (WACC - g)
   = $0.39B × (1 + 2%) / (12% - 2%)
   = $0.40B / 0.10
   = $4.0B
```

## 九、学习要点总结

### 核心公式

```
1. FCF = 净利润 × 40%（欣兴 Capex 较高）
2. PV = FCF / (1+WACC)^n
3. TV = FCF₅ × (1+g) / (WACC-g)
4. DCF = Σ PV + TV现值
```

### 关键差异（vs 台积电）
