## 一、麦肯锡分析框架 (Mckinsey)

### 1.1 7S 模型 (McKinsey 7S Framework)

| 维度 | 含义 | 检查要点 |
|------|------|----------|
| **Strategy（战略）** | 公司发展方向 | 竞争优势、目标市场、差异化 |
| **Structure（结构）** | 组织架构 | 汇报关系、部门设置 |
| **Systems（系统）** | 流程与系统 | 决策流程、信息系统 |
| **Shared Values（价值观）** | 企业文化 | 核心价值观、使命 |
| **Style（风格）** | 管理风格 | 领导方式、决策模式 |
| **Staff（人才）** | 人才队伍 | 人才配置、能力发展 |
| **Skills（技能）** | 核心能力 | 关键技能、竞争优势 |

### 1.2 麦肯锡问题分析

**金字塔原理 (Minto Pyramid Principle)**：

```
结论先行 → 以上统下 → 归类分组 → 逻辑递进

结构：
- 核心论点（最顶层）
- 论据1、论据2、论据3（支撑核心论点）
- 每个论据下有子论据（支撑论据）
```

**表达顺序**：
1. 结论（Summary）
2. 关键论点（Key Arguments）
3. 支撑数据（Supporting Data）

### 1.3 麦肯锡五力分析 (Porter's Five Forces)

| 力 | 含义 | 关键问题 |
|------|------|----------|
| **供应商议价能力** | 上游供应商的谈判力 | 供应商集中度、替代品 |
| **客户议价能力** | 下游客户的谈判力 | 客户集中度、替代品 |
| **新进入者威胁** | 新竞争者进入的难度 | 进入壁垒、规模经济 |
| **替代品威胁** | 替代产品/服务的威胁 | 替代品价格/性能 |
| **同业竞争程度** | 现有竞争者的激烈程度 | 行业集中度、差异化 |

### 1.4 麦肯锡价值链分析

```
基础活动：
├── 进料后勤（Inbound Logistics）
├── 生产运营（Operations）
├── 出货后勤（Outbound Logistics）
├── 营销与销售（Marketing & Sales）
└── 服务（Service）

支持活动：
├── 企业基础设施（Firm Infrastructure）
├── 人力资源管理（Human Resource Management）
├── 技术开发（Technology Development）
└── 采购（Procurement）
```

## 二、波士顿矩阵 (BCG Matrix)

### 2.1 四象限分类

| 象限 | 名称 | 市场增长率 | 市场份额 | 策略 |
|------|------|------------|----------|------|
| **Stars** | 明星 | 高 | 高 | 投资增长 |
| **Cash Cows** | 现金牛 | 低 | 高 | 收割利润 |
| **Question Marks** | 问题 | 高 | 低 | 选择性投资 |
| **Dogs** | 瘦狗 | 低 | 低 | 出售/淘汰 |

### 2.2 半导体行业应用

| 公司/产品 | 定位 | 策略 |
|-----------|------|------|
| **NVIDIA H100** | Stars | 投资增长，扩大领先 |
| **台积电代工** | Stars | 投资产能，满足 AI 需求 |
| **传统 CPU** | Cash Cows | 稳定利润，控制成本 |
| **成熟制程** | Cash Cows | 稳定现金流 |
| **新兴 AI 芯片** | Question Marks | 评估是否值得投资 |
| **传统封装** | Dogs | 考虑退出 |

### 2.3 组合管理策略

```
Stars → 投资（维持/扩大市场份额）
Cash Cows → 收割（最大化现金流）
Question Marks → 选择（投资有潜力的，出售无潜力的）
Dogs → 出售/淘汰（剥离非核心业务）
```

## 五、SWOT 分析升级版

### 5.1 TOWS 矩阵

| 战略类型 | 战略 |
|----------|------|
| **SO（优势-机会）** | 利用优势抓住机会 |
| **WO（劣势-机会）** | 克服劣势抓住机会 |
| **ST（优势-威胁）** | 利用优势应对威胁 |
| **WT（劣势-威胁）** | 最小化劣势避免威胁 |

### 5.2 半导体公司 SWOT 示例

**以欣兴电子为例**：

**Strengths（优势）**：
- ABF 载板技术领先
- NVIDIA/AMD 核心供应商
- 台系管理效率高

**Weaknesses（劣势）**：
- 2024 年营收下滑 34.6%
- 客户集中度较高
- 资本支出压力大

**Opportunities（机会）：
- AI GPU 需求爆发
- CoWoS 产能扩张
- 国产替代趋势

**Threats（威胁）**：：
- 玻璃基板技术替代
- 中国厂商竞争
- 行业周期波动

**战略建议**：
- SO：扩大 ABF 产能，抓住 AI 机遇
- WO：拓展客户，降低集中度
- ST：加速技术研发，应对替代威胁
- WT：控制成本，提升良率

## 七、实战案例：半导体公司分析框架

### 7.1 分析大纲

```
一、行业分析
   1.1 市场规模与增长
   1.2 产业链结构
   1.3 竞争格局
   1.4 五力分析

二、公司分析
   2.1 商业模式
   2.2 竞争优势（价值链分析）
   2.3 财务表现
   2.4 SWOT 分析

三、估值分析
   3.1 DCF 估值
   3.2 同业比较
   3.3 目标价

四、投资建议
   4.1 投资要点
   4.2 风险提示
   4.3 操作建议
```

### 7.2 关键问题清单

**行业层面**：
- 行业增长驱动力是什么？
- 竞争格局如何？CR5 是多少？
- 进入壁垒是什么？
- 产业链话语权如何分配？

**公司层面**：
- 公司竞争优势是什么？（技术/成本/规模/渠道）
- 客户结构如何？集中度？
- 毛利率/净利率趋势？
- 资本支出计划？

**估值层面**：
- 当前估值在历史什么位置？
- 与同业相比贵还是便宜？
- DCF 内在价值是多少？
- 有什么未充分反映的利好/利空？

## 八、学习总结

### 核心框架

1. **麦肯锡 7S**：组织诊断

```
问题定义 → 信息收集 → 分析框架 → 假设验证 → 结论建议
```

- 结论先行
- 逻辑清晰
- 数据支撑
- 结构完整
