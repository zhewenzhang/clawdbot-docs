## 三、Kirin 9010/9020 详解

### 3.1 Kirin 9010 规格

| 参数 | 规格 | 说明 |
|------|------|------|
| **制程** | 7nm (SMIC N+2) | 国产化 |
| **CPU** | 2+6+4 架构 | 超大核+性能核+能效核 |
| **超大核** | Cortex-X2 @2.62GHz | 性能核心 |
| **性能核** | Cortex-A710 x6 | 中核 |
| **能效核** | Cortex-A510 x4 | 小核 |
| **GPU** | Maleoon 910 | 自研 GPU |
| **AI** | 达芬奇架构 | 8 TOPS |
| **Modem** | 巴龙 5000 | 5G |
| **内存** | LPDDR5 | 最大 16GB |

### 3.2 Kirin 9020 规格

| 参数 | 规格 | 提升 |
|------|------|------|
| **制程** | 7nm+ (SMIC N+2) | 优化 |
| **CPU** | Cortex-X3 + A715 + A510 | 架构升级 |
| **频率** | 3.0 GHz (X3) | +15% |
| **GPU** | Maleoon 920 | +20% |
| **AI** | 15 TOPS | +80% |
| **Modem** | 巴龙 5000+ | 优化 |

### 3.3 性能对比
