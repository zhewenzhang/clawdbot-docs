================================================================================
                    第一部分：台积电（TSM）估值报告
================================================================================

一、公司概况

台积电（Taiwan Semiconductor Manufacturing Company）是全球最大的晶圆代工厂，
市场份额超过 60%。公司成立于 1987 年，总部位于台湾新竹。

二、核心数据

| 指标 | 数值 |
|------|------|
| 市值 | $800B |
| P/E | 25x |
| 2024 营收增长 | +25% |
| 毛利率 | 53% |
| 营业利润率 | 42% |

三、营收预测（Top-down）

1. 全球代工市场（2024）：$150B
2. 台积电市场份额：60%
3. 目标营收：$90B

四、DCF 估值

假设：
- 营收增长（2024-2028）：15%
- 毛利率：52-54%
- WACC：9%
- 永续增长率：3%

估值结果：
- 内在价值：$469B
- 当前市值：$800B
- 估值溢价：41%

五、投资建议

评级：持有
理由：估值偏高，但 AI 需求强劲，长期增长确定

================================================================================
                    第二部分：欣兴电子（3037.TW）估值报告
================================================================================

一、公司概况

欣兴电子是全球最大的 IC 载板供应商，主要产品包括 ABF 载板、BGA 载板等，
客户包括英特尔、NVIDIA、AMD 等。

二、核心数据

| 指标 | 数值 |
|------|------|
| 市值 | $15B |
| P/E | 18x |
| 2024 营收增长 | +30% |
| 毛利率 | 25% |
| ABF 产能 | 30K/月 |

三、营收预测（Bottom-up）

1. ABF 载板需求：$5B（2024）
2. 欣兴市场份额：20%
3. ABF 营收：$1B
4. 其他产品：$2B
5. 总营收：$3B

四、DCF 估值

假设：
- 营收增长（2024-2028）：20%
- 毛利率：25-28%
- WACC：10%
- 永续增长率：3%

估值结果：
- 内在价值：$2.32B
- 当前市值：$3.23B
- 估值溢价：28%

五、投资建议

评级：持有（关注回调）
理由：估值偏高，但 AI 需求驱动增长确定

                    第三部分：中芯国际（688981.SS）估值报告
================================================================================

一、公司概况

中芯国际是中国最大的晶圆代工厂，提供 0.35μm 至 14nm 制程代工服务，
受美国制裁影响，先进制程发展受限。

二、核心数据

| 指标 | 数值 |
|------|------|
| 市值 | $50B |
| P/E | 50x（亏损，PB 2.5x） |
| 2024 营收增长 | +20% |
| 毛利率 | 20% |
| 产能利用率 | 85% |

三、营收预测（Bottom-up）

1. 成熟制程产能：800K/月
2. 产能利用率：85%
3. ASP：$1,500
4. 营收：$1.2B

四、SOTP 估值

1. 成熟制程业务：$1B（10x P/B）
2. 先进制程期权：$5B（技术突破期权价值）
3. 合计：$6B

五、投资建议

评级：关注
理由：国产替代核心标的，但短期受制裁影响
