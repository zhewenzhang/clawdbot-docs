## 行为改变

### 改变1：主动汇报习惯
```
之前：学了不主动说，等你问
之后：学完立即汇报（5分钟内）
```

### 改变2：CAPDCA检查清单
```
收到任务后立即问：
1. 这是复杂任务吗？（>3步骤）
2. 需要什么Skill？
3. 我有足够信息吗？
4. 可能出错的地方？
```

### 改变3：结论必须有证据
```
之前：「应该可以」「大概对」
之后：运行XX命令 → 输出是YY → 结论是ZZ
```
