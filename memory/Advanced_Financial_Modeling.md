## 一、营收建模方法

### 1.1 Top-down 方法

```
营收 = 市场规模 × 市场份额

示例（台积电 2025E）：
- AI 芯片市场规模：$500B
- 台积电代工份额：90%
- 台积电 AI 营收 = $500B × 90% = $450B ❌（过高）

修正：
- 代工占芯片成本：10%
- 代工市场：$500B × 10% = $50B
- 台积电份额：60%
- 台积电 AI 营收 = $50B × 60% = $30B
```

### 1.2 Bottom-up 方法

```
营收 = 产能 × 利用率 × ASP × 良率

示例（欣兴 ABF 载板）：
- 设计产能：100 万平方米/年
- 利用率：85%
- ASP：$3,000/平方米
- 良率：80%
- 营收 = 100 × 85% × $3,000 × 80% = $204M/年
```

### 1.3 营收拆分模板

| 产品线 | 销量 | ASP | 营收 | YoY |
|--------|------|-----|------|-----|
| AI 芯片载板 | 70 万片 | $4,500 | $315M | +50% |
| 服务器载板 | 50 万片 | $2,800 | $140M | +30% |
| 消费电子 | 100 万片 | $1,500 | $150M | +10% |

## 二、成本结构分析

### 2.1 成本分解

| 成本项目 | 占比 | 特点 |
|----------|------|------|
| **直接材料** | 40-50% | 与营收成正比 |
| **直接人工** | 15-20% | 相对稳定 |
| **制造费用** | 20-30% | 固定成本为主 |
| **折旧摊销** | 10-15% | 非现金支出 |

### 2.2 毛利率驱动因素

```
毛利率 = 1 - COGS / 营收

影响毛利率的因素：
1. 产品组合（高端产品毛利率更高）
2. 规模效应（产能利用率提升）
3. 定价能力（供需紧张时提价）
4. 成本优化（材料成本下降）
5. 良率提升（良率从 85% → 90%）
```

### 2.3 半导体公司成本特征

| 公司类型 | 毛利率 | 研发费用率 | Capex/营收 |
|----------|--------|------------|------------|
| **IC 设计** | 50-70% | 20-30% | 5-10% |
| **晶圆代工** | 45-55% | 8-12% | 30-50% |
| **封装测试** | 20-30% | 3-5% | 10-15% |
| **IC 载板** | 20-30% | 5-8% | 15-25% |

## 三、Capex 分析

### 3.1 Capex 周期

| 周期阶段 | Capex 特征 | 原因 |
|----------|------------|------|
| **上行初期** | 开始增加 | 需求增长，产能不足 |
| **上行高峰** | 达到峰值 | 抢产能、抢份额 |
| **下行初期** | 开始下降 | 需求放缓 |
| **下行底部** | 降至低位 | 产能过剩，谨慎投资 |

### 3.2 Capex 效率指标

| 指标 | 计算 | 含义 |
|------|------|------|
| **Capex/营收** | Capex / 营收 | 投资强度 |
| **Capex/折旧** | Capex / D&A | 产能扩张速度 |
| **产能增量/Capex** | 新增产能 / Capex | 投资效率 |
| **投资回收期** | Capex / FCF | 回收时间 |

### 3.3 产能扩张计划

## 四、营运资本分析

### 4.1 营运资本公式

```
营运资本 = 应收账款 + 存货 - 应付账款

净营运资本变动 = 本期营运资本 - 上期营运资本
```

### 4.2 关键周转天数

| 指标 | 计算 | 含义 | 健康范围 |
|------|------|------|----------|
| **DSO** | 应收账款 / 营收 × 365 | 回款天数 | 30-60 天 |
| **ITO** | 存货 / COGS × 365 | 存货周转 | 60-90 天 |
| **DPO** | 应付账款 / COGS × 365 | 付款天数 | 30-60 天 |
| **CCC** | DSO + ITO - DPO | 现金循环周期 | 30-60 天 |

### 4.3 半导体行业特征

| 公司类型 | DSO | ITO | DPO | CCC |
|----------|-----|-----|-----|-----|
| **IC 设计** | 45 天 | 90 天 | 30 天 | 105 天 |
| **晶圆代工** | 30 天 | 60 天 | 45 天 | 45 天 |
| **封装测试** | 45 天 | 30 天 | 45 天 | 30 天 |

## 五、估值方法进阶

### 5.1 分部估值（SOTP）

```
适用场景：多元化业务公司

示例（英特尔）：
┌────────────────┬──────────┬─────────┬──────────┐
│ 业务部门       | 估值方法 | 估值    | 依据     │
├────────────────┼──────────┼─────────┼──────────┤
│ PC 处理器      | P/E 10x  | $80B    | 成熟业务 │
│ 数据中心       | P/E 20x  | $60B    | 成长业务 │
│ 代工业务       | EV/EBITDA| $30B    | 亏损业务 │
│ Mobileye       | P/S 15x  | $20B    | 独立估值 │
├────────────────┼──────────┼─────────┼──────────┤
│ **合计**       | -        | **$190B** | -       │
└────────────────┴──────────┴─────────┴──────────┘
```

### 5.2 EV/EBITDA vs P/E

| 维度 | EV/EBITDA | P/E |
|------|-----------|-----|
| **适用** | 重资产公司 | 轻资产公司 |
| **优点** | 剔除非现金影响 | 简单直观 |
| **缺点** | 忽略研发费用 | 忽略资本结构 |
| **半导体** | 代工、封测 | IC 设计 |

### 5.3 PEG 估值

```
PEG = P/E / 盈利增长率

解读：
- PEG < 1：相对低估
- PEG = 1：合理估值
- PEG > 1：相对高估

示例（NVIDIA）：
- P/E = 50x
- 盈利增长率 = 50%
- PEG = 50 / 50 = 1.0（合理）
```

## 六、DCF 模型进阶

### 6.1 多阶段 DCF

```
阶段 1（1-3年）：详细预测
- 逐年预测营收、成本、Capex
- 计算 FCF

阶段 2（4-5年）：简化假设
- 营收增长率逐年递减
- 利润率逐步稳定

阶段 3（第6年起）：永续增长
- 永续增长率 g
- 终值计算
```

### 6.2 终值敏感性

|----------|------------------|------|
| 第 5 年 | 8x | $3.2B |

| 第 5 年 | 12x | $4.8B |

### 6.3 DCF 估值检查清单

| 检查项 | 核查要点 |
|--------|----------|
| **营收假设** | 是否与行业增速一致？ |
| **毛利率假设** | 是否高于/低于历史？ |
| **Capex 假设** | 是否与产能扩张匹配？ |
| **WACC 假设** | β 是否合理？ |
| **永续增长率** | 是否合理（< GDP 增速）？ |
| **敏感性** | 关键假设变化的影响？ |

## 八、学习要点总结

### 8.1 核心公式

```
1. 营收 = 产能 × 利用率 × ASP × 良率
2. 毛利率 = 1 - COGS / 营收
3. CCC = DSO + ITO - DPO
4. FCF = 净利润 + D&A - Capex - ΔWC
5. PEG = P/E / 盈利增长率
6. SOTP = 各业务估值之和
```

### 8.2 估值方法选择

| 公司类型 | 推荐方法 | 关键指标 |
|----------|----------|----------|
| **IC 设计** | P/E、PEG | 营收增速、研发费用 |
| **晶圆代工** | EV/EBITDA、DCF | 产能利用率、Capex |
| **封装测试** | EV/EBITDA | 产能利用率、良率 |
| **IC 载板** | P/E、DCF | ABF 份额、AI 需求 |

### 8.3 财务分析框架

```
1. 理解业务 → 2. 拆分假设 → 3. 建立模型 → 4. 估值计算 → 5. 敏感性分析 → 6. 投资建议
```
