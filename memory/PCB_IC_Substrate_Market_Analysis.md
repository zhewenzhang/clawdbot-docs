### 二、产品分类：ABF vs BT 载板

#### ABF 载板（高性能计算）
- **FC PGA/LGA/BGA**：用于 CPU、GPU、ASIC 等大芯片
- **特点**：大尺寸、高层数（12-32层）、细线路
- **市场占比**：2024年 50.4%，2029年预计 56.7%
- **增速**：CAGR 12.4%（载板中最快）

#### BT 载板（消费/移动端）
| 产品 | 应用 | 特点 | CAGR |

| **WB PBGA/CSP** | 消费电子 | 打线封装 | 4.1% |
| **Module** | RF 模块、SiP | 多芯片集成 | 8.2% |

## 📊 第三部分：产品尺寸与叠构规格

### 一、PCB 产品尺寸与规格

| 产品类别 | 典型尺寸 | 层数 | 线宽/线距 | 应用 |
|----------|----------|------|-----------|------|
| **Commodity** | 多样 | 2-8 层 | >100μm | 家电、消费电子 |
| **Multilayer** | 多样 | 4-24 层 | 50-100μm | 工业、汽车 |
| **HDI** | 多样 | 4-16 层 | 40-75μm | 手机、平板 |
| **Package Substrate** | 110x110mm | 8-32 层 | 9-25μm | CPU、GPU、AI 芯片 |
| **Flex** | 柔性 | 1-6 层 | 25-75μm | 可穿戴、折叠屏 |

### 二、IC 载板尺寸与叠构

#### ABF 载板（FC PGA/LGA/BGA）

| 规格 | 参数 |
|------|------|
| **典型尺寸** | 110x110mm（最大可达 120x120mm） |
| **层数** | 8-32 层（AI 芯片可达 20+ 层） |
| **线宽/线距** | 9/9μm → 5/5μm → 3/3μm（演进中） |
| **铜厚** | 12-18μm |
| **表面处理** | ENIG、OSP |
| **叠构** | 1+N+1, 2+N+2, 3+N+3（增层结构） |
| **核心材料** | ABF 薄膜（味之素） |
| **应用** | CPU、GPU、FPGA、ASIC、AI 加速器 |

#### BT 载板（FCCSP/FC-DRAM）

| 规格 | 参数 |
|------|------|
| **典型尺寸** | 40x40mm - 80x80mm |
| **层数** | 4-10 层 |
| **线宽/线距** | 15-30μm |
| **核心材料** | BT 树脂（双马来酰亚胺三嗪） |
| **应用** | 手机 SoC、存储芯片、RF 模块 |

#### Module 载板

| 规格 | 参数 |
|------|------|
| **典型尺寸** | 20x20mm - 50x50mm |
| **层数** | 2-8 层 |
| **特点** | 多芯片集成（SiP） |
| **应用** | WiFi 模块、电源模块、传感器模块 |

### 三、技术演进路线

| 年份 | ABF 线宽/线距 | ABF 层数 | 载板尺寸 |
|------|---------------|----------|----------|
| 2022 | 9/9μm | 12-16 层 | 110x110mm |
