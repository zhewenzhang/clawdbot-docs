## 四、蒙特卡洛模拟基础

### 4.1 模拟原理

```
输入变量分布：
- 营收增速：正态分布（均值 20%，标准差 10%）
- 毛利率：三角分布（悲观 35%，基准 40%，乐观 45%）
- WACC：均匀分布（8%-12%）

模拟次数：10,000 次
输出：估值分布、VaR（风险价值）
```

### 4.2 模拟结果解读

| 统计量 | 含义 | 应用 |
|--------|------|------|
| **均值** | 预期估值 | 投资决策参考 |
| **中位数** | 中间估值 | 去除极端值影响 |
| **标准差** | 估值波动 | 风险衡量 |
| **VaR 95%** | 5% 概率跌破 | 下行风险 |
| **VaR 99%** | 1% 概率跌破 | 极端风险 |

## 六、投资组合风险管理

### 6.1 风险指标

| 指标 | 计算 | 含义 |
|------|------|------|
| **VaR 95%** | 95% 置信度下的最大损失 | 日度风险 |
| **CVaR** | 超过 VaR 的平均损失 | 尾部风险 |
| **最大回撤** | 历史最高到最低 | 历史风险 |
| **夏普比率** | (收益 - 无风险) / 波动率 | 风险调整收益 |

## 七、情景规划实战

### 7.1 案例：2025 年 AI 芯片投资

**乐观情景**（25% 概率）：

- AI 需求持续爆发，营收 +50%
- 毛利率提升至 45%
- P/E 扩张至 35x
- **估值**：当前 +40%

**基准情景**（50% 概率）：

- AI 需求稳健，营收 +25%
- 毛利率 40%
- P/E 维持 22x
- **估值**：当前水平

**悲观情景**（25% 概率）：

- AI 热潮消退，营收 +10%
- 毛利率下降至 35%
- P/E 收缩至 15x
- **估值**：当前 -35%

### 7.2 加权期望估值

```
期望估值 = 25% × (+40%) + 50% × (0%) + 25% × (-35%)
        = 10% + 0% - 8.75%
        = +1.25%

结论：风险调整后期望收益约 1.25%，风险收益比不佳
```

## 八、学习要点总结

### 8.1 核心概念

| 概念 | 含义 | 应用 |
|------|------|------|
| **情景分析** | 多假设下的估值区间 | 投资决策 |
| **敏感性分析** | 单变量变化影响 | 关键假设识别 |
| **蒙特卡洛模拟** | 多变量随机模拟 | 风险量化 |
| **VaR** | 风险价值 | 尾部风险管理 |
| **尾部风险** | 小概率大损失事件 | 极端情况准备 |

### 8.2 分析框架

```
1. 定义情景（乐观/基准/悲观）
2. 设定假设（增长、毛利率、P/E）
3. 计算各情景估值
4. 加权期望估值
5. 识别关键敏感变量
6. 制定对冲/风险管理
```

### 8.3 风险控制原则

1. **分散化**：不要把所有资金放在一个标的
2. **止损**：设定最大亏损阈值
3. **对冲**：期权保护或空头对冲
4. **现金管理**：保持一定现金比例
5. **动态调整**：根据市场变化调整配置
