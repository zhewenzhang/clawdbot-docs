## 🔧 五、关键公式

### 市场分析

```
载板需求 = AI芯片出货量 × CoWoS产能利用率 × 单芯片载板价值

CAGR = (终值/初值)^(1/年数) - 1

市场份额 = 公司营收 / 市场规模 × 100%
```

### 产能计算

```
月度产能 = 日产能 × 工作天数 × 良率

CoWoS产能(K/月) = 晶圆产能 × 载板良率
```

## 🎯 八、分析框架速记

### 需求端分析

```
AI芯片出货量 → CoWoS产能 → ABF载板需求
手机出货量 → FCCSP需求 → BT载板需求
汽车渗透率 → 车用载板需求
```

### 供给端分析

```
产能利用率 = 实际产出 / 设计产能
扩产计划 → 未来供给增量
良率提升 → 有效产能增加
```
