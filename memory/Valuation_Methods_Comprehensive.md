## 一、估值方法概述

### 1.1 估值方法分类

```
估值方法

├── 绝对估值法
│   ├── DCF（折现现金流）
│   ├── 股息折现模型（DDM）
│   └── 剩余收益模型（RIM）
│
└── 相对估值法
    ├── P/E（市盈率）
    ├── P/B（市净率）
    ├── P/S（市销率）
    ├── EV/EBITDA
    └── PEG（市盈率增长）

└── 特殊方法
    ├── SOTP（分部估值）
    ├── 清算价值
    └── 重置成本
```

### 1.2 方法对比

| 方法 | 优点 | 缺点 | 适用场景 |
|------|------|------|----------|
| **DCF** | 理论基础强 | 假设敏感 | 稳定增长公司 |
| **P/E** | 简单直观 | 易受会计影响 | 盈利稳定公司 |
| **EV/EBITDA** | 剔除资本结构 | 计算复杂 | 重资产公司 |
| **PEG** | 考虑成长性 | 单一指标 | 成长股 |
| **SOTP** | 考虑多元化 | 主观性强 | 多元化公司 |

## 二、DCF 估值法深度分析

### 2.1 DCF 原理

**DCF = 未来现金流折现**

```
DCF 公式

          CF1        CF2        CF3        CFn       TV
Value = ─────── + ─────── + ─────── + ... + ─────── + ───────
        (1+r)¹     (1+r)²     (1+r)³           (1+r)ⁿ   (1+r)ⁿ

其中：
- CF = 各期现金流
- r = 折现率（WACC）
- TV = 终值（Terminal Value）
- n = 预测期
```

### 2.2 DCF 建模步骤

```
DCF 建模步骤

Step 1: 历史分析
├── 分析历史营收、毛利率、费用率
├── 识别关键驱动因素
└── 建立预测假设

Step 2: 营收预测
├── Top-down 或 Bottom-up 建模
├── 5-10 年营收预测
└── 验证合理性

Step 3: 利润预测
├── 毛利率假设
├── 费用率假设
└── 净利润预测

Step 4: 自由现金流计算
├── FCF = EBIT × (1-tax) + D&A - Capex - NWC
├── 计算各年 FCF
└── 验证合理性

Step 5: WACC 计算
├── WACC = E/V × Ke + D/V × Kd × (1-tax)
├── 计算各组成部分
└── 选择合适参数

Step 6: 终值计算
├── 永续增长法：TV = FCF × (1+g) / (WACC - g)
├── EV/EBITDA 法：TV = EBITDA × Multiple
└── 交叉验证

Step 7: 估值计算
├── 各年 FCF 折现
├── 终值折现
├── 计算企业价值
├── 减去净债务
└── 计算股权价值
```

## 三、WACC 计算深度分析

### 3.1 WACC 公式

```
WACC = E/V × Ke + D/V × Kd × (1 - Tax)

其中：
- E/V = 股权价值/企业价值（股权比例）
- D/V = 债务价值/企业价值（债务比例）
- Ke = 股权成本
- Kd = 债务成本
- Tax = 所得税率
```

### 3.2 股权成本（Ke）计算

**CAPM 模型**

```
Ke = Rf + β × (Rm - Rf) + RP

其中：
- Rf = 无风险利率
- β = Beta（系统性风险）
- (Rm - Rf) = 市场风险溢价
- RP = 特殊风险溢价（国家风险、行业风险等）
```

**Ke 计算示例**

| 参数 | 数值 | 说明 |
|------|------|------|
| **无风险利率（Rf）** | 4.0% | 10 年国债收益率 |
| **Beta（β）** | 1.2 | 公司相对市场波动 |
| **市场风险溢价（Rm-Rf）** | 5.0% | 历史平均 |
| **国家风险溢价** | 1.0% | 中国公司特殊 |
| **Ke** | 11.0% | 4% + 1.2×5% + 1% |

### 3.3 债务成本（Kd）计算

**Kd 参考**

| 信用评级 | 债务成本 | 说明 |
|----------|----------|------|
| **AAA** | 4.0% | 最高信用 |
| **AA** | 4.5% | 高信用 |
| **A** | 5.0% | 良好信用 |
| **BBB** | 5.5% | 一般信用 |
| **BB** | 7.0% | 次级债 |
| **B** | 9.0% | 高收益债 |

### 3.4 WACC 计算示例

**案例：某 IC 载板公司**

| 参数 | 数值 | 说明 |
|------|------|------|
| **股权比例（E/V）** | 70% | 资本结构 |
| **债务比例（D/V）** | 30% | 资本结构 |
| **股权成本（Ke）** | 11.0% | CAPM 计算 |
| **债务成本（Kd）** | 5.5% | A 级信用债 |
| **所得税率（Tax）** | 25% | 公司税率 |
| **WACC** | 8.9% | 70%×11% + 30%×5.5%×(1-25%) |

## 四、相对估值法深度分析

### 4.1 P/E（市盈率）

**公式**

```
P/E = 股价 / 每股收益（EPS）
```

**示例**

| 公司 | 股价 | EPS（2024E） | P/E | 行业平均 |
|------|------|--------------|-----|----------|
| **欣兴** | $15.0 | $0.83 | 18x | 20x |
| **南电** | $8.0 | $0.50 | 16x | 20x |
| **Ibiden** | ¥3,000 | ¥150 | 20x | 20x |

**P/E 估值**

```
目标股价 = EPS × 目标 P/E

欣兴目标股价：
$0.83 × 20 = $16.6（当前 $15.0，上涨 11%）
```

### 4.2 EV/EBITDA

**公式**

```
EV/EBITDA = 企业价值 / EBITDA
```

**示例**

| 公司 | EV | EBITDA（2024E） | EV/EBITDA | 行业平均 |
|------|-----|-----------------|-----------|----------|
| **欣兴** | $16B | $1.0B | 16x | 15x |
| **南电** | $9B | $0.6B | 15x | 15x |
| **Ibiden** | $10B | $0.7B | 14x | 15x |

**EV/EBITDA 估值**

```
目标 EV = EBITDA × 目标 EV/EBITDA
目标股价 = (目标 EV - 净债务) / 股份数

欣兴目标股价：
$1.0B × 15 - $2B = $13B EV
$13B / 1B 股 = $13.0（当前 $15.0，下跌 13%）
```

### 4.3 PEG（市盈率增长）

**公式**

```
PEG = P/E / 盈利增长率（G）

PEG < 1：低估
PEG = 1：合理
PEG > 1：高估
```

**示例**

| 公司 | P/E | 2024 增速 | 2025 增速 | 平均增速 | PEG |
|------|-----|-----------|-----------|----------|-----|
| **欣兴** | 18x | +25% | +30% | 27.5% | 0.65 |
| **南电** | 16x | +20% | +25% | 22.5% | 0.71 |
| **Ibiden** | 20x | +10% | +12% | 11.0% | 1.82 |

**PEG 估值**

```
目标 P/E = G × 1.0（假设 PEG=1）

欣兴目标 P/E = 27.5 × 1.0 = 27.5x
目标股价 = $0.83 × 27.5 = $22.8（当前 $15.0，上涨 52%）
```

### 4.4 P/B（市净率）

**公式**

```
P/B = 股价 / 每股净资产（BPS）
```

**示例**

| 公司 | 股价 | BPS | P/B | 行业平均 |
|------|------|-----|-----|----------|
| **欣兴** | $15.0 | $5.0 | 3.0x | 3.5x |
| **南电** | $8.0 | $3.5 | 2.3x | 3.5x |

## 五、SOTP 分部估值法

### 5.1 SOTP 原理

**SOTP = Sum of The Parts**

```
公司价值 = 各业务价值之和

SOTP 适用于：
- 多元化公司
- 不同业务估值方法不同
- 各业务相对独立
```

### 5.2 SOTP 示例

**案例：某多元化半导体公司**

**业务分部**

| **PCB** | $0.5B | P/E | 12x | $6B |

**SOTP 估值**

| 项目 | 数值 |
|------|------|
| **IC 载板** | $15B |

### 6.1 估值结果对比

|----------|------|----------|----------|

### 6.2 估值区间

```
估值区间分析

          估值方法对比
              ↑
   $4.00 ──┼────────● PEG（高估）
           │
   $3.50 ──┼────● SOTP
           │
   $3.00 ──┼─● P/E
           │
   $2.50 ──┼───● EV/EBITDA
           │
   $2.33 ──┼─────● DCF
           │
   $2.00 ──┼───────────────────────────
           │
   $1.00 ──┼───────────────────────────
           └──────────────────────────→
              当前股价 $3.05

估值区间：$2.33 - $4.00
中值：$3.07
当前股价：$3.05（合理区间）
```

| 因素 | 影响 | 调整幅度 |
|------|------|----------|
| **增长加速** | 估值上调 | +10-20% |

## 七、学习要点总结

### 7.1 核心公式

| 方法 | 公式 | 关键参数 |
|------|------|----------|
| **DCF** | Σ FCF / (1+WACC)ⁿ | FCF、WACC、g |
| **WACC** | E/V×Ke + D/V×Kd×(1-T) | Ke、Kd、资本结构 |
| **P/E** | 股价 / EPS | 盈利增长 |
| **EV/EBITDA** | EV / EBITDA | 资本结构 |
| **PEG** | P/E / G | 成长性 |

### 7.2 估值步骤

1. **选择方法**：根据公司特点选择
2. **计算参数**：WACC、增长率等
3. **计算估值**：应用公式
4. **交叉验证**：多方法对比
5. **调整估值**：考虑特殊因素
6. **得出结论**：给出估值区间

### 7.3 常见错误

| 错误 | 避免方法 |
|------|----------|
| **WACC 错误** | 仔细计算各组成部分 |
| **增长率假设过高** | 保守假设 |
| **忽视风险** | 进行敏感性分析 |
| **单一方法** | 多方法交叉验证 |
| **忽视周期性** | 周期位置调整 |
