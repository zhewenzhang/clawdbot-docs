## 七、学习要点总结

### 7.1 核心概念

| 概念 | 含义 | 投资意义 |
|------|------|----------|
| **利率周期** | 央行政策利率的周期性变化 | 决定估值水平 |
| **汇率周期** | 货币相对价值的周期性变化 | 影响进出口 |
| **通胀周期** | 物价水平的周期性变化 | 影响成本和需求 |
| **周期叠加** | 多周期同时作用 | 放大效应 |

### 7.2 关键规律

1. **利率领先**：利率变化领先经济周期 6-12 个月
2. **汇率滞后**：汇率变化滞后利率 3-6 个月
3. **通胀滞后**：通胀变化滞后经济 3-6 个月
4. **周期叠加**：多重周期叠加会放大波动

### 7.3 投资原则

1. **关注周期位置**：当前处于什么周期阶段
2. **关注领先指标**：用领先指标预判周期
3. **分散风险**：不同周期资产配置
4. **动态调整**：根据周期变化调整配置
