## 三、DCF 估值核心公式

### 3.1 自由现金流 (FCF)

```
FCF = 净利润 + 折旧摊销 - 资本支出 - 营运资本增加

简化：FCF = 净利润 × (1 - Capex/净利润比例)
```

### 3.2 DCF 估值模型

```
企业价值 = Σ [FCFₜ / (1+WACC)ᵗ] + TV / (1+WACC)ⁿ

其中：
- FCF = 自由现金流
- WACC = 加权平均资本成本
- TV = 终值（Terminal Value）
- t = 年份
- n = 预测期
```

### 3.3 WACC 计算

```
WACC = (E/V) × Ke + (D/V) × Kd × (1 - 税率)

其中：
- E = 股权市值
- D = 债务市值
- V = 企业价值 (E+D)
- Ke = 股权成本（CAPM）
- Kd = 债务成本
```

### 3.4 CAPM 模型（股权成本）

```
Ke = Rf + β × (Rm - Rf)

其中：
- Rf = 无风险利率（10年国债）
- β = 贝塔系数
- Rm = 市场收益率
- (Rm - Rf) = 股权风险溢价
```

### 3.5 终值计算

**永续增长法**：
```
TV = FCFₙ × (1 + g) / (WACC - g)
```

**退出倍数法**：
```
TV = EBITDAₙ × 退出倍数
```

## 四、估值倍数速查

### 4.1 常用估值指标

| 指标 | 计算 | 含义 | 适用场景 |
|------|------|------|----------|
| **P/E** | 市值 / 净利润 | 市盈率 | 成熟期公司 |
| **EV/EBITDA** | 企业价值 / EBITDA | 企业价值倍数 | 重资产行业 |
| **P/S** | 市值 / 营收 | 市销率 | 亏损公司 |
| **P/B** | 市值 / 账面价值 | 市净率 | 资产密集型 |
| **EV/Revenue** | 企业价值 / 营收 | 营收倍数 | SaaS/平台 |

### 4.2 半导体行业估值

| 环节 | P/E 范围 | EV/EBITDA 范围 | 合理 P/E |
|------|----------|----------------|----------|
| **IC 设计** | 20-50x | 12-25x | 25-35x |
| **晶圆代工** | 15-25x | 8-15x | 18-22x |
| **封装测试** | 12-20x | 6-12x | 14-18x |
| **IC 载板** | 10-25x | 8-15x | 12-18x |
| **设备** | 15-30x | 10-20x | 18-25x |

### 4.3 估值比较法

```
相对估值 = 目标公司 × (可比公司 P/E)

步骤：
1. 选择可比公司（同业、业务相似）
2. 计算可比公司 P/E 中位数
3. 应用到目标公司净利润
4. 得到目标市值

注意：调整成长性差异（PEG）
PEG = P/E / 盈利增长率
PEG < 1：相对低估
PEG > 1：相对高估
```

## 六、ROE 分解（杜邦分析）

### 6.1 ROE 三因子

```
ROE = 净利率 × 资产周转率 × 杠杆系数

ROE = (净利润/营收) × (营收/资产) × (资产/权益)
```

### 6.2 杜邦分析示例

| 公司 | ROE | 净利率 | 资产周转 | 杠杆 | 分析 |
|------|-----|--------|----------|------|------|
| **NVIDIA** | 45% | 35% | 0.8 | 1.5 | 高净利率驱动 |
| **台积电** | 25% | 35% | 0.5 | 1.5 | 净利率 + 杠杆 |
| **英特尔** | 10% | 15% | 0.4 | 1.8 | 全面下滑 |
| **AMD** | 30% | 10% | 0.9 | 3.0 | 杠杆驱动 |

## 九、学习要点总结

### 9.1 核心公式

```
1. FCF = 净利润 + D&A - Capex - 营运资本增加
2. DCF = Σ FCF / (1+WACC)^t + TV / (1+WACC)^n
3. WACC = (E/V)×Ke + (D/V)×Kd×(1-T)
4. Ke = Rf + β×(Rm-Rf)
5. ROE = 净利率 × 资产周转率 × 杠杆
6. P/E = 市值 / 净利润
7. PEG = P/E / 盈利增长率
```

### 9.2 估值框架

| 方法 | 适用 | 优点 | 缺点 |
|------|------|------|------|
| **DCF** | 稳定现金流 | 内在价值 | 假设敏感 |
| **P/E** | 成熟期 | 简单直观 | 忽略增长 |
| **EV/EBITDA** | 重资产 | 剔除非现金 | 忽略研发 |
| **PEG** | 成长股 | 考虑增长 | 需要盈利 |

### 9.3 分析顺序

```
1. 理解业务 → 2. 分析周期 → 3. 财务分析 → 4. 估值 → 5. 决策
```
