## 数据库结构进度

### 已设计字段

#### 基础信息（8字段）
```
company_id    - 公司ID（唯一标识）
company_name_cn  - 公司中文名
company_name_en  - 公司英文名
stock_code    - 股票代码
exchange      - 上市交易所
founded_year  - 成立年份
headquarters  - 总部位置
website       - 官网
```

#### 芯片类别（3字段）
```
chip_type     - 芯片类型（主分类）
chip_subtype  - 子类型
application   - 应用领域
```

#### 载板使用分析（5字段）
```
abf_usage     - ABF载板使用（是/否/潜在/未知）
bt_usage      - BT载板使用（是/否/潜在/未知）
packaging_tech - 封装技术
chip_size     - 芯片尺寸
est_annual_volume - 预计年用量（Kpcs）
```

#### 深度分析（6字段）
```
main_customers - 主要客户
foundry       - 晶圆代工
assembly_test  - 封装测试厂
tech_node     - 技术节点
revenue_usd   - 营收规模（百万美元）
growth_outlook - 增长预期（高/中/低/稳定）
```

#### 状态管理（4字段）
```
data_status   - 数据状态
last_verified - 最后验证日期
source        - 数据来源
notes         - 备注
```
