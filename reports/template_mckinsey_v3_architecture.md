## 一、设计理念

### 1.1 目标读者
- **角色**：芯片载板市场开发专家
- **需求**：评估芯片设计公司的载板开发价值
- **决策**：是否值得投入资源开发该客户

### 1.2 麦肯锡风格核心原则

**金字塔原理（Pyramid Principle）**
```
        结论（是否值得开发）
           /    |    \
      理由1   理由2   理由3
      / | \   / | \   / | \
    数据支撑（财务/技术/市场）
```

**MECE原则（Mutually Exclusive, Collectively Exhaustive）**
- 互斥：各维度分析不重叠
- 穷尽：覆盖所有关键因素

**数据驱动洞察（Data-Driven Insights）**
- 每个结论都有数据支撑
- 避免主观判断，用"So What?"深挖价值
