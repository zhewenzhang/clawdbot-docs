# Clawdbot 文档库

本仓库存储 Clawdbot 的文档和数据文件。

## 访问说明

### 数据仪表板
- **地址**: https://zhewenzhang.github.io/clawdbot-docs/dashboard.html
- **状态**: 需要在仓库设置中启用 GitHub Pages

### 启用 GitHub Pages
1. 进入 [Repository Settings](https://github.com/zhewenzhang/clawdbot-docs/settings/pages)
2. Source: **Deploy from a branch**
3. Branch: **main** / (root)
4. 点击 **Save**

### 文件列表

| 文件 | 说明 |
|------|------|
| dashboard.html | 数据仪表板主页 |
| excel_index.json | Excel 文件索引 |

## 自动同步

本仓库通过 GitHub Actions 或手动脚本自动同步。
