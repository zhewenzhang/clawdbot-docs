# 8️⃣ 存储芯片格局：DRAM 和 NAND

> 三星、SK Hynix、美光的三角竞争

---

## 一、两种存储芯片

### 1.1 DRAM

**全称**：Dynamic Random Access Memory

| 特点 | 说明 |
|-----|------|
| **易失性** | 断电数据丢失 |
| **速度** | 极快 (ns 级) |
| **用途** | 电脑/手机内存 |
| **代表** | DDR4/DDR5 |

### 1.2 NAND Flash

| 特点 | 说明 |
|-----|------|
| **非易失性** | 断电数据保留 |
| **速度** | 快 (μs 级) |
| **用途** | SSD、U盘、存储卡 |
| **代表** | 3D NAND |

### 1.3 对比

| 维度 | DRAM | NAND |
|-----|------|------|
| 速度 | 更快 | 快 |
| 成本 | 高 | 低 |
| 密度 | 低 | 高 |
| 功耗 | 高 | 低 |
| 用途 | 内存 | 存储 |

## 四、技术演进

### 4.1 DRAM

```
DDR4 (2014) → DDR5 (2020) → DDR6 (2025+)
  ↓              ↓             ↓
 1.6GT/s      6.4GT/s      12.8GT/s
```

### 4.2 NAND

```
SLC → MLC → TLC → QLC → PLC
  ↓      ↓      ↓      ↓      ↓
 1bit   2bit   3bit   4bit   5bit
 寿命↓        密度↑
```

### 4.3 3D NAND

```
层数：64 → 128 → 232 → 500+
     ↓      ↓      ↓      ↓
  2017   2020   2023   2026+
```
