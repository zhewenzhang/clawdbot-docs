# 🚗 汽车芯片崛起

> 从"四个轮子一个沙发"到"四个轮子一台电脑"

---

## 一、一场汽车革命

### 1.1 变化有多大？

| 维度 | 传统汽车 | 智能汽车 |
|-----|---------|---------|
| **代码行数** | 1亿行 | 10亿+行 |
| **芯片数量** | 几十颗 | 上百颗 |
| **成本占比** | <20% | >50% |
| **创新点** | 机械 | 电子+软件 |

**汽车正在从"机械产品"变成"电子产品"。**

### 1.2 一个比喻

```
传统汽车 = 功能手机 (打电话为主)
智能汽车 = 智能手机 (功能丰富)
```

## 二、汽车芯片分类

### 2.1 按功能分

| 类型 | 作用 | 代表芯片 |
|-----|------|---------|
| **计算芯片** | 自动驾驶、智能座舱 | 地平线 J5/J6 |
| **功率芯片** | 电机控制、电源管理 | IGBT、SiC MOSFET |
| **传感芯片** | 雷达、摄像头、激光雷达 | 图像传感器 |
| **通信芯片** | 车联网、V2X | 以太网芯片 |
| **存储芯片** | 数据存储 | DRAM、NAND |
| **模拟芯片** | 信号处理 | 电源管理 IC |

### 2.2 按价值分

| 芯片类型 | 价值占比 | 增长趋势 |
|---------|---------|---------|
| **计算芯片** | 30% | ↑↑↑ (智驾+座舱) |
| **功率芯片** | 25% | ↑↑ (电动化) |
| **传感芯片** | 15% | ↑ (智驾) |
| **通信芯片** | 10% | ↑↑ (车联网) |
| **存储芯片** | 10% | ↑ (数据) |
| **模拟芯片** | 10% | → (稳定) |

## 四、计算架构演进

### 4.1 分布式 → 域集中 → 中央计算

```
阶段 1 (2020): 分布式
├── 智驾域控制器
├── 座舱域控制器
├── 底盘域控制器
└── 各自独立

阶段 2 (2023): 域集中
├── 智驾+座舱融合
├── 底盘+车身融合
└── 3-4 个域控制器

阶段 3 (2025+): 中央计算
├── 1 个中央大脑
├── 区域控制器
└软件定义汽车
```

### 4.2 特斯拉架构

```
中央计算平台 (FSD + 娱乐)
    ↓
    ├→ 前车身控制器
    ├→ 左车身控制器
    ├→ 右车身控制器
    └→ 后车身控制器
```

## 八、总结

### 核心要点

1. **汽车芯片价值占比从 20% → 50%+**
2. **智驾芯片：NVIDIA 领先，地平线追赶**
3. **座舱芯片：高通垄断，国产突破**
4. **功率芯片：SiC 快速增长，比亚迪/英飞凌主导**
5. **激光雷达：中国厂商全球领先**
6. **架构演进：分布式 → 域集中 → 中央计算**
