# 2️⃣ 什么是先进封装？

> 解读 CoWoS、InFO、EMIB 等先进封装技术

---

## 一、为什么需要先进封装？

### 1.1 摩尔定律放缓

**传统路线**：缩小晶体管 → 越来越难

| 问题 | 说明 |
|-----|------|
| 物理极限 | 原子级别，无法再小 |
| 成本飙升 | 先进制程动辄百亿美元 |
| 良率下降 | 缺陷概率指数增加 |

### 1.2 先进封装=新答案

**换道超车**：用封装技术突破瓶颈

```
制程不够？ → 封装来凑
    ↓
晶体管密度 = 制程密度 + 封装密度
```

## 二、两大封装类型

### 2.1 2.5D 封装

**原理**：芯片平铺在基板上，用硅桥连接

```
     芯片 A      芯片 B
       │           │
       └─────┬─────┘
             │
        ┌────▼────┐
        │  硅桥   │  ← 高速通道
        └────┬────┘
             │
        ┌────▼────┐
        │   基板  │  ← 承载芯片
        └─────────┘
```

### 2.2 3D 封装

**原理**：芯片垂直堆叠

```
   芯片 A  ← 顶层
   ──────
   芯片 B  ← 中层
   ──────
   芯片 C  ← 底层
```

## 三、主流技术对比

### 3.1 CoWoS (台积电)

**全称**：Chip on Wafer on Substrate

| 特点 | 说明 |
|-----|------|
| **厂商** | 台积电 |
| **代表客户** | NVIDIA (H100)、AMD |
| **应用** | AI 训练芯片 |

**结构**：
```
芯片 → 硅中介层 → 基板
```

### 3.2 InFO (台积电)

**全称**：Integrated Fan-Out

| 特点 | 说明 |
|-----|------|
| **厂商** | 台积电 |
| **代表客户** | Apple (A 系列) |
| **应用** | 手机处理器 |

**优势**：不需要硅桥，成本更低

### 3.3 EMIB (Intel)

**全称**：Embedded Multi-die Interconnect Bridge

| 特点 | 说明 |
|-----|------|
| **厂商** | Intel |
| **代表产品** | Ponte Vecchio |
| **应用** | 数据中心 GPU |

**优势**：灵活连接不同芯片

## 六、总结

### 技术对比表

| 技术 | 厂商 | 类型 | 应用 |
|-----|------|------|------|
| CoWoS | 台积电 | 2.5D | AI 芯片 |
| InFO | 台积电 | 2.5D | 手机 |
| EMIB | Intel | 2.5D | 数据中心 |
| SoIC | 台积电 | 3D | 高性能 |

### 核心要点

1. **2.5D = 平铺 + 硅桥**
2. **3D = 垂直堆叠**
3. **CoWoS = AI 芯片标配**
4. **中国封装正在追赶**
