# 9️⃣ 光刻机是什么？

> 芯片制造的核心设备，"半导体工业的明珠"

---

## 一、光刻机的地位

**没有光刻机，就没有芯片。**

```
硅片 → 光刻 → 刻蚀 → 沉积 → 封装
  ↓
光刻是最关键的环节
```

### 1.1 芯片制造流程

```
1. 硅片清洗
2. 氧化
3. **光刻** ← 光刻机
4. 刻蚀
5. 离子注入
6. 薄膜沉积
7. 抛光
8. 测试
```

---

## 二、光刻原理

### 2.1 比喻

光刻 = "芯片印刷术"

```
掩模板 (底片)
    ↓
紫外光
    ↓
光刻胶 (感光材料)
    ↓
显影 (冲洗)
    ↓
电路图案
```

### 2.2 关键参数

| 参数 | 说明 |
|-----|------|
| **光源波长** | 193nm、13.5nm |
| **分辨率** | 越小越先进 |
| **套刻精度** | 多层对齐精度 |
| **产能** | 每小时晶圆数 |

---

## 三、光刻机类型

### 3.1 分类

| 类型 | 光源 | 波长 | 适用制程 |
|-----|------|------|---------|
| **DUV** | 准分子激光 | 193nm | 7nm-28nm |
| **EUV** | 极紫外光 | 13.5nm | 5nm-3nm |
| **High-NA EUV** | 极紫外光 | 13.5nm | 2nm以下 |

### 3.2 价格对比

| 类型 | 价格 | 代表厂商 |
|-----|------|---------|
| DUV | $1000-5000万 | ASML、尼康、佳能 |
| EUV | $1.5-2亿美元 | ASML 独家 |
| High-NA EUV | $$3-4亿美元 | ASML |

---

## 四、EUV：为什么这么难？

### 4.1 技术挑战

| 挑战 | 说明 |
|-----|------|
| **光源** | 极紫外光极难产生 |
| **反射镜** | 精度要求 < 0.1nm |
| **真空** | 工作在真空环境 |
| **产能** | 每小时处理 100+ 片 |

### 4.2 ASML 垄断

```
EUV 技术：ASML 100% 垄断
原因：
1. 专利壁垒
2. 供应链复杂
3. 投资巨大
```

**一台 EUV，10 万个零件。**

## 五、中国光刻机现状

### 5.1 上海微电子 (SMEE)

| 产品 | 进展 | 制程 |
|-----|------|------|
| **90nm** | 已量产 | 90nm |
| **28nm** | 研发中 | 28nm |
| **14nm** | 计划中 | 14nm |

### 5.2 差距

| 类型 | ASML | 中国 |
|-----|------|------|
| EUV | ✅ 量产 | ❌ 禁运 |
| DUV | ✅ 7nm | 28nm |
| 浸没式 | ✅ 先进 | 追赶中 |

### 5.3 卡脖子

```
EUV 设备 → 荷兰 ASML (美国施压)
     ↓
   无法获取
     ↓
 先进制程 (7nm以下) → 受限
```

## 六、光刻机未来

### 6.1 技术演进

```
DUV (193nm) → EUV (13.5nm) → High-NA EUV → Hyper-NA EUV
    ↓              ↓              ↓             ↓
  28nm+         5nm-7nm        2nm          1nm以下
```

### 6.2 替代技术

| 技术 | 说明 | 进展 |
|-----|------|------|
| **纳米压印** | 用模具压印 | 研发中 |
| **电子束直写** | 用电子束刻画 | 产量低 |
| **X射线光刻** | X射线光源 | 实验阶段 |

## 七、总结

### 核心要点

1. **光刻机 = 芯片制造核心设备**
2. **EUV = 先进制程唯一选择**
3. **ASML = EUV 100% 垄断**
4. **中国光刻机 = 28nm 量产，EUV 受限**
5. **差距大，但持续追赶**

| 类型 | 波长 | 制程 | 价格 | 厂商 |
|-----|------|------|------|------|
| DUV | 193nm | 7nm-28nm | $1000-5000万 | ASML/尼康/佳能 |
| EUV | 13.5nm | 5nm-3nm | $1.5-2亿 | ASML |
| High-NA | 13.5nm | 2nm | $3-4亿 | ASML |
