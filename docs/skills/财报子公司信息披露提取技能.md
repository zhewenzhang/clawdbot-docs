## 一、技能概述

### 1.1 什么是"子公司信息披露"？

台湾上市公司年报中，"子公司信息披露"是法定的信息披露章节，通常包含：
- 公司组织架构图（持股关系）
- 各子公司基本信息（成立时间、地址、注册资本）
- 主要业务和产品
- 董事长/总经理信息
- 各子公司财务概况

### 1.2 为什么这个技能重要？

1. **投资分析**：了解集团结构和各子公司贡献
2. **风险评估**：识别对特定子公司的依赖度
3. **业务理解**：掌握公司的产能布局和地理分布
4. **竞品分析**：对比不同公司的子公司策略

## 三、提取技能要点

### 3.1 PDF文本提取命令

```bash
# 提取所有文本
pdftotext filename.pdf - | grep -i "subsidiary\|affiliate"

# 提取特定章节
pdftotext -f 100 -l 120 filename.pdf - | head -100

# 提取最后几页
pdftotext -layout filename.pdf - | tail -500

# 搜索关键词
pdftotext filename.pdf - | grep -i "China\|Mainland"
```

## 四、Python自动化工具

### 4.1 PDF文本提取

```python
import subprocess

def extract_pdf_text(pdf_path, pages=None):
    """提取PDF文本"""
    if pages:
        result = subprocess.run(
            ['pdftotext', '-f', str(pages[0]), '-l', str(pages[1]), pdf_path, '-'],
            capture_output=True, text=True
        )
    else:
        result = subprocess.run(
            ['pdftotext', '-layout', pdf_path, '-'],
            capture_output=True, text=True
        )
    return result.stdout
```

### 4.2 子公司信息提取

```python
import re

def extract_china_subsidiaries(text):
    """提取大陆子公司信息"""
    # 搜索包含 "China" 或 "Kunshan" 等关键词的段落
    pattern = r'Unimicron Technology[^\n]*\n[^\n]*China[^\n]*\n[^\n]*US\$[0-9,]+'
    
    matches = re.findall(pattern, text)
    return matches
```
