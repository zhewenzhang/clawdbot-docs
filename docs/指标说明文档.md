# 指标说明文档

## 资金流向指标

### 1. 主力净流入 (Main Net Inflow)

**定义:** 中单+大单+特大单的净买入金额

**计算公式:**
```
主力净流入 = (中单买入 + 大单买入 + 特大单买入) - (中单卖出 + 大单卖出 + 特大单卖出)
```

**判断标准:**
| 数值 | 含义 |
|-----|------|
| > 1亿 | 主力大幅流入 |
| > 0 | 主力净流入 |
| < 0 | 主力净流出 |
| < -1亿 | 主力大幅流出 |

### 2. 散户净流入 (Retail Net Inflow)

**定义:** 小单的净买入金额

**计算公式:**
```
散户净流入 = 小单买入 - 小单卖出
```

**判断标准:**
| 数值 | 含义 |
|-----|------|
| > 0 | 散户净买入 |
| < 0 | 散户净卖出 |
| < -5000万 | 散户大幅抛售 |

### 3. 资金集中度 (Concentration)

**定义:** 主力净流入占总净流入的比例

**计算公式:**
```
资金集中度 = 主力净流入 / (主力净流入 + 散户净流入)
```

**判断标准:**
| 数值 | 含义 |
|-----|------|
| > 80% | 资金高度集中 |
| 50%-80% | 资金偏集中 |
| < 50% | 资金分散 |

### 4. 资金连续性 (Continuity)

**定义:** N日内净流入为正的天数比例

**计算公式:**
```
资金连续性 = 正流入天数 / 统计天数
```

**判断标准:**
| 数值 | 含义 |
|-----|------|
| > 80% | 资金持续流入 |
| > 50% | 资金偏流入 |
| < 50% | 资金偏流出 |

### 5. 净流入率 (Flow Rate)

**定义:** 净流入金额占成交额的比例

**计算公式:**
```
净流入率 = 净流入金额 / 成交额
```

**判断标准:**
| 数值 | 含义 |
|-----|------|
| > 5% | 资金大幅流入 |
| > 1% | 资金温和流入 |
| < -5% | 资金大幅流出 |

## 市场情绪指标

### 北向资金 (North Money)

**定义:** 通过沪深港通从香港流入大陆股市的资金

**判断:**
- 净流入 > 0: 外资看多
- 净流入 > 10亿: 大幅看多
- 净流出 < 0: 外资看空

### 南向资金 (South Money)

**定义:** 通过沪深港通从大陆流入香港股市的资金

**判断:**
- 净流入 > 0: 内资港股配置增加
- 持续净流出: 可能存在资金外流压力

### 市场情绪综合判断

| 北向 | 南向 | 市场情绪 |
|-----|------|---------|
| 大幅流入 | 流入 | 强烈看多 |
| 流入 | 流出 | 偏多 |
| 流出 | 流入 | 偏空 |
| 大幅流出 | 流出 | 强烈看空 |

## 告警规则

### 严重告警 (Critical)
- 主力净流入 > 3亿
- 主力净流出 < -3亿

### 警告告警 (Warning)
- 主力净流入 > 1亿
- 散户净流出 < -5000万
- 资金集中度 > 80%

### 信息提示 (Info)
- 资金集中度变化 > 30%
- 主力净流入变化 > 50%
