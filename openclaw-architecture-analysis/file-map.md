## 📌 使用建议

### 新安装 OpenClaw 时
1. 按优先级顺序配置
2. 先配置 P0 级文件
3. 再配置 P1 级文件

### 日常使用时
1. 定期更新 MEMORY.md
2. 参考 AGENTS.md 做决策
3. 使用 HEARTBEAT.md 保持节奏

### 优化使用时
1. 定期清理 memory/ 归档
2. 更新 Skills 使用策略
3. 优化心跳检查配置
